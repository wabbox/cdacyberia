RAPPEL :  Avez vous bien imprimé toutes les dernières mises à jour ????
 
INFORMATIONS

Rappel : Toutes ces évolutions ne sont pas accessibles en Version 6

 

1°)  BALANCE GENERALE

Une nouvelle rubrique dans le grand livre et la balance : Niveau de regroupement.
Elle vous permet d'éditer la balance avec des sous totaux
- Niveau 1 : Vous obtenez les sous totaux de la classe
- Niveau 2 : Vous obtenez les sous totaux de la sous classe

 

2°) EXPORT SOUS EXCEL

Les options recherches et états vous permettent dorénavant d'exporter directement sur
Excel.  Après avoir fait votre requête, le logiciel vous propose 3 solutions : Imprimer le résultat, exporter sous format txt ou exporter sous excel.
Si vous choisissez cette dernière option, le nouvel écran qui s'affiche vous permet de créer directemet votre tableau. 
Il suffit de préciser dans la rubrique  Exporter la table, le nom de la table. Tous les champs de la table concernée vont s'afficher.  Double cliquez sur les champs voulus pour créer votre tableau.

Pour supprimer une colonne : Selectionner la colonne et cliquez sur la petite flèche ascendante rouge.  Pour supprimer toutes les colonnes, cliquez sur la double flèche rouge.

Créer votre tableau puis cliquez sur exporter et donnez un nom à votre fichier.
Il suffit ensuite de l'ouvrir avec Excel.

3°)  FICHE CLIENT : Plan d'accès

Nouvelle option dans la fiche client :  En cliquant sur plan d'accès, vous obtenez le plan du quartier où est domicilié votre client. Vous pouvez ensuite faire votre itinéraire.

4°)  AGENDA : Gestion des contacts

Vous pouvez maintenant imprimer la liste des contacts et supprimer un ou plusieurs contacts.

 

5°)  CONTRAT : Affichage des taux santé - Prévoyance

Si vous saisissez les taux, dans le contrat, en imprimant la fiche client et en visualisant ses contrats, les taux seront affichés.

6°)  ETATS RECAPITULATIFS : Sinistres ouverts

Vous pouvez maintenant demander d'afficher d'imprimer les sinsitres ouverts pour une période donnée.

7°)  COMPTE CLIENT : Modification de présentation du compte client

Nous avons modifié la présentation du compte client : le mode de règlement y figure (chèque,virement... )ainsi que le montant.

 

8°)  GRAND LIVRE : Consultation directe d'une écriture

Lorsque vous êtes dans le grand Livre : 1 compte - détail -  En double cliquant sur la ligne concernée vous accédez directement à l'écriture.
