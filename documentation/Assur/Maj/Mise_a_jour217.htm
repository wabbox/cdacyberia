EVOLUTIONS

         1°) FICHE CONTRAT

3 nouvelles rubriques ont été ajoutées sur la fiche contrat :
-  A réviser le :
- Revision demandée le :
- Révision reçue le :

Ces rubriques concernent les contrats qui doivent être révisés annuellement.
En saisissant une date, dans la rubrique à réviser le,  au jour J, cela apparaitra dans l'agenda.
Vous ferez alors,un courrier à votre client  et dans le contrat, vous saisissez la date d'envoi du courrier, dans rubrique  revision demandée le.
Lorsque le client, aura renvoyé ces renseignements, compléter la fiche contrat avec la date de réception des pièces dans la rubrique révision reçue le.

Dans les paramètres généraux,   définissez le délai entre votre demande et le délai de retour des pièces (exemple 20 pour 20 jours)  dans la rubrique Délai retour révision
