EVOLUTIONS

         1°) ECRAN DE SAISIE CLIENT

Saisie "assistée" pour le pays - ville et code postal.

Si vous saisissez les 2 premiers chiffres du code postal, le logiciel vous affiche la liste des codes postaux. débutant par les deux premiers chiffres saisis. Cela est valable pour 1 chiffre ou plusieurs chiffres.

Cette saisie assistée est valable egalement pour le pays et la ville.

1°) SAISIE D'UN VEHICULE  sur l'écran de la saisie de la carte grise.

La saisie des informations de la carte grise a été modifiée. Le modèle est avant la marque.
Vous pouvez maintenant PARAMETRER vos véhicules et les marques en PARAMETRES  GESTION DES ENUMERES.

Vous avez une rubrique Auto Marques dans la liste des énumérés

a) Saisir d'abord les valeurs de l'énuméré qui correspond à la Marque du véhicule
Nous en avons saisi quelques unes ;  à vous de continuer.........

b) pour la saisie des modèles

Vous devez affecter le modèle à la Marque.  La Marque est considérée comme le Parent.
Lors de la saisie de vos modèles, affecter  toujours dans la rubrique PARENT, le code de la marque.

Ci-dessous, un exemple.

L'énuméré Auto Marque est toujours sélectionné. Cliquer sur le bouton MODIFIER de la partie ENUMERE.  Un message vous informe "Une modification d'un énuméré peut affecter le fonctionnement d'Assur" Cliquer sur Continuer et au message suivant confirmez vous : cliquer modifier.  Vous arrivez sur un écran composé d'un tableau de 5 colonnes
code valeur - Parent - Rang - Valeur et ancien code.

La marque possède un code valeur.  Exemple : Audi  416  et Mercedes 418
Pour la marque Audi, vous voulez rajouter les modèles : A3 - A 4 - A 5
Double cliquer sur la dernière ligne blanche de la liste. Sur le nouvel écran  saisissez dans la rubrique Valeur  A3   et dans la rubrique Parent  416  (vous affectez le modèle A3) au "Parent" Audi qui a le code 416.

  1°) SUPPRESSION PROGRESSIVE DU DOUBLE AFFICHAGE

Progressivement, le double affichage (FRF - EUR) sera supprimé. ATTENTION pour les utilisateurs qui font de la pollicitation  verifier que vos paramétrages de pollicitation sont en EUR et on en FRF  (devise, le montant de la prime et de la com). Si vous êtes en FRF, faites les modifications nécessaires pour avoir votre pollicitation en Euros.
