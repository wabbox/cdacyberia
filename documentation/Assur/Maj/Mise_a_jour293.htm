INFORMATIONS

Rappel : Certaines évolutions ne sont pas accessibles en Version 6

 

1°)  EDI NSA

Une nouvelle fonctionnalité existe pour les utilisateurs travaillant avec NSA : les EDI

L'objectif de cet EDI : vous éviter la double saisie sur le système NSA et sur Assur

http://www.cda-informatique.fr/Documentation/Assur/General/EDI_NSA.htm

Coût du module : 230,00 Euros TTC

2°)  RENUMEROTATIONS CONTRATS CONTINENT en GENERALI

 Les utilisateurs qui travaillaient avec Continent ont reçu de Generali la nouvelle numérotation des contrats.  Afin de vous éviter de faire manuellement les modifications :

- Demander à la compagnie LA TABLE DES CORRESPONDANCES entre N° continent et N° générali. Cette table vous est adressée par mail.
Nous la transmettre

- Après étude de votre fichier, nous vous adresserons le programme correspondant.

Vous pourrez ainsi en automatique, modifier les N° de contrat.

  3°)  ECHEANCIER DES PRELEVEMENTS

Une nouvelle option dans le Menu quittances. Cette option vous permet d'éditer et d'informer vos clients qui sont en prélèvement, sur l'échéancier de leur prélèvements.

Conditions préalables à l'édition de cet échéancier : les primes, dans les contrats, doivent être à jour. Par ailleurs, les échéances sont calculées à partir de la rubrique fractionnement du contrat et non pas à partir de la rubrique prélèvements dans la fiche contrat.

 4°)  DELETTRAGE à partir d'une certaine date

Il est possible maintenant de delettrer à partir d'une certaine date

 5°)  ACCES DIRECT A LA DOCUMENTATION UTILISATEUR

Pour visualiser le compte du client, cliquer sur le rectangle bleu COMPTE au dessus de la rubrique solde.

Le bouton Compte est remplacé dans la fiche client par  "I"  pour information.  Ce bouton vous permettra d'accéder directement à la documentation utilisateur.
Il faut préciser dans le logiciel le chemin d'accès à la documentation.
Aller En paramètres  paramètres généraux  Cliquer sur le bouton Acces et  dans la rubrique Chemin d'aide, saisir le chemin suivant :


www.cda-informatique.fr/documentation/assur/general/

Nous allons donc, petit à petit, mettre sur chaque écran, un bouton donnant accès à la documentation utilisateur--->  Un peu de patience et des changements dans les écrans.
