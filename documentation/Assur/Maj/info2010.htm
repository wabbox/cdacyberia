INFORMATIONS AUX UTILISATEURS DE LA  VERSION 7

QUI FONT UNE MISE A JOUR  XP

Page précédente
RAPPEL :  Avez vous bien imprimé toutes les dernières mises à jour ????
 

 

Si vous êtes utilisateur de la version 7 et que votre système d'exploitation est XP :
Ne faites pas la mise à jour automatique XP.
Celle ci entre en conflit avec 4D et Assur.

Si vous l'avez faite, lorsque vous lancez Assur, vous avez en arrière plan une fenetre avec le nom de votre ordinateur ou de votre Cabinet.   Certains fonctions d'E Assur vous font quitter le logiciel.

Pour régulariser :

En démarrer - Paramètres - Panneau de configuration  (ou démarrer  Panneau de configuration)
Cliquer Ajout - suppression de programme - sélectionner Windows XP Hot Fix - KB824141 et cliquer sur supprimer (remove).  Accepter les différents messages des dialogues.

Puis redemarrer votre ordinateur.
