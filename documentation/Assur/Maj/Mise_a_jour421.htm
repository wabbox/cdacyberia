INFORMATIONS

  RAPPEL : La documentation utilisateur est accessible via l'option Aide dans chaque Menu du logiciel.  Si vous ne disposez pas de cette option en Utilisateur, demander au Super_utilisateur de vous l'affecter.                              

1°)  LISTE DES CAS IDA - Avec Images

Dans la mise à jour 420, nous avons mis à votre disposition la liste des cas IDA (format texte)

Le nouveau format vous permet d'avoir également les images.

1°) Si vous n'avez pas encore intégré les Cas IDA

En paramètres, une nouvelle fonction  Cas IDA vous permet d'importer le fichier des cas IDA
Saisir un cas  valider  vous avez ensuite  le bouton recevoir  cliquez dessus et enregistrer sur le bureau le fichier IDA (cliq droit souris  enregistrer sous). Le fichier est compacté. Cas_IDA.exe.
Sur le bureau, Double cliquez dessus et le fichier se décompacte dans C:\EDI.

Puis cliquez importer  et localiser le fichier  dans C:\EDI et l'ouvrir. Dans la nouvelle fenetre,  selectionnez dans la table d'import IDA (à la place d'interface)  et vous avez le tableau qui apparait.  Cliquez sur IMporter. Le fichier est importé.
Supprimer votre saisie initiale en sélectionnant la ligne et  Voir et cliquez sur l'icone de la poubelle pour le détruire.   Certains cas IDA sont agrémentés d'image.  Double cliquez sur la ligne et vous aurez le descriptif ainsi que l'image corresondante.

Il vous suffira ensuite, en saisie sinistre   de cliquer sur le bouton IDA pour obtenir la liste.

2°) Si vous avez déjà  intégré les Cas IDA

Vous avez donc la liste mais sans les images.
   - Soit en Paramètres - Cas IDA, vous détruisez chaque ligne  puis vous vous référer au 1°)
et ainsi vous aurez la liste des Cas IDA avec image,

   - Soit vous conservez cette liste mais sans les images.
