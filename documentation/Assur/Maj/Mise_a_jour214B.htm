EVOLUTIONS

         1°) ECRAN SPECIFIQUE EN FONCTION DES BRANCHES

De nouveaux écrans ont été créés (retraite collective, santé collective.....)

Pour bénéficier de ces nouvelles branches, cliquez ici

2°) UTILISATION DE L'ICONE   X   POUR FERMER UN ECRAN DE CONSULTATION

Lorsque vous êtes en consultation client - contrat - véhicules, vous pouvez, pour revenir directement au menu général (sans utiliser les boutons quitter) cliquer sur la  X de la fenetre active.

3°) EN COURS DE CREATION

Pour les contrats collectifs (retraite, prévoyance, santé, flotte....), 3 innovations importantes

- Vous pourrez importer dans le logiciel, la liste des adhérents.....

- Plus de double ou triple saisie dans les autres contrats collectifs Vous pourrez affecter directement la liste des adhérents.   Exemple : une entreprise souscrit une prévoyance groupe avec X adhérents. Puis un contrat santé groupe pour les mêmes adhérents.
Vous n'aurez plus à saisir, sur ce nouveau contrat, les adhérents. En cliquant sur le bouton Affecter, les adhérents seront automatiquement créés sur ce nouveau contrat.

- 3 critères de recherche sont à votre disposition : N° SS, N° adhérent, Nom adhérent.
