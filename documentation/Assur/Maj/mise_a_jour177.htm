1) Rappel 1:

Nous avons eu a traiter plusieurs incidents concernant les sauvegardes. Certains de vos fournisseurs de matériel créent des procédures de sauvegarde automatique. Il arrive que ces sauvegardes ne fonctionnent pas correctement. Dans ce cas, cela signifie que s'il y a un probleme sur votre ordinateur, vous perdez toutes vos données.

VERIFIEZ QUE VOS SAUVEGARDES SONT VALIDES

 

2) Rappel 2 :

 Dans les documents que nous vous adressons, certaines expressions ou certaines phrases apparaissent en couleur bleue,soulignées. Cela signifie que c'est un lien vers un document qui se trouve sur notre site.

Cliquez sur cette expression ou cette phrase pour pouvoir lire le document et l'imprimer.

 

3) Evolutions

a) Modification des etats :

Dans Etats -----> Etats récapitulatifs , les états de quittances étaient imprimés triés sur le numero de contrat.

On peut désormais demander qu'ils soient triés sur le nom du client.

b) Nouvelle Fonctionnalité :

Il est désormais possible d'effectuer des règlements de sinistres par virements automatiques

Pour lire et imprimer la Documentation cliquez ici : Documentation

c) Encaissement de quittance : on peut désormais encaisser une quittance par carte bancaire. Le numero de carte fait l'objet d'un controle de validité.

d) Evolution des prélèvements : deux modifications ont été effectuées. Tout d'abord, les prélèvements qui soldent une quittance sont comptabilisés directement en solde cette quittance, et non pas en acompte comme auparavant. Par ailleurs, il est possible de comptabiliser automatiquement les impayés.

Pour lire et imprimer la Documentation, cliquez ici : Documentation
