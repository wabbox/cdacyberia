INFORMATIONS

Rappel : Certaines évolutions ne sont pas accessibles en Version 6

 

1°)  RELEVES ET MEMOS 

Ces documents apparaissent désormais dans la liste GED du client ou dans Contacts

2°)  ENVOI DE MAIL à partir d'un COURRIER

 Le courrier 4D Write est transformé en une page HTML. Avant d'être envoyé, les images éventuelles contenues dans le courrier ne sont pas envoyés.
Si l'image est dans la page : pas de problème de création du mail, l'image est ignorée.
Si l'image est dans le texte : il faut impérativement avoir installé Quicktime pour pouvoir envoyer le mail. 4D Write extrait l'image du texte, crée un fichier .jpg et crée un lien dans le texte sur cette image.
Dans une mise à jour ultérieure, les images dans la page ou dans le texte seront jointes au mail en pièces jointes.

3°)  GESTION DES COLLABORATEURS

En gestion des collaborateurs, 2 nouvelles options :

- synthèse des bordereaux : liste des apporteurs avec le total des retrocessions pour la période
- syntèse des termes : liste des apporteurs avec le total des émissions pour la période.

4°)  MODIFICATIONS DIVERSES

Corrections des bugs : envois de fonds flottes auto -  generation de quittances non confiées
lectures des documents en pdf en V7 et V8
