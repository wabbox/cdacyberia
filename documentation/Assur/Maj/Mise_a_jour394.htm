INFORMATIONS

1°)  ADDITIF :  AVIS ECHEANCE avec Directives Européennes

Dans la mise à jour 389 - Point 1 Avis d'échéance, ne pas Tenir compte du Point B.

Le taux de TVA à paramétrer dans les paramètres généraux  ne concerne que la TVA "Cabinet".
Le taux de TVA compagnie, est à saisir DANS LA QUITTANCE

2°)  ENVOIS DE FONDS

Vous pouvez faire figurer sur vos envois de fonds un pied de page.
Ce dernier est à paramétrer dans Paramètres  paramètres généraux  Sélectionner l'entité  Modifier entité et cliquez sur le bouton en bas de l'écran : pied de page.

3°)  Courrier ou Mail à une adresse différente

Lors de l'envoi d'un courrier ou d'un mail par la fonction Courrier  Modifier et envoyer, il vous est possible désormais de sélectionner le destinataire (exemple une compagnie, et un intercoluteur dans la compagnie - ou un apporteur).

Ainsi le courrier sera adressé à ce destinataire là.
Pour cela il faut paramétrer vos courriers avec une variable et non plus avec une insertion de champs ( ex client titre, client nom...)

Les variables sont :  Menu Insertion  Expression 4D - cliquez sur l'onglet expression  cliquez sur modifier et dans la rubrique blanche saisir   v_adr_1  cliquez OK  OK.  Les autres variables sont

v_adr_2
v_adr_3
v_adr_4
v_adr_5

4°)  MAILING avec Insertion de champs dans les fonctions Recherches et Etats

Dans la fonction mailing, les critères demandés vous permettent d'obtenir une liste de clients.
Dans le paramétrage du courrier on ne peut pas insérer de champs spécifiques. En effet, lorsqu'un client à plusieurs contrats, comment le logiciel peut il savoir qu'il s'agit de ce contrat ou de ce contrat là.

Par contre dans la fonction Recherches et Etats de chaque Option de Gestion, vous avez la fonction mailing. Celle ci est disponible après avoir fait votre requête et après avoir cliqué sur le bouton envoyer vers.  Cocher Mailing.
Attention : le courrier s'imprimera autant de fois que demandé.
Exemple : Mailing sur les contrats  Si le client à 3 contrats répondant à votre critère de recherche, le logiciel imprimera 3 fois le courrier.
