EVOLUTIONS

         1°) FICHE CLIENT

En cliquant sur voir  Sinistres, vous pouvez maintenant imprimer la liste des sinistres clients pour une période donnée.

2°) SAISIE DES CONTRATS   Flotte automobile Multi Compagnies

Il est possible de gérer une flotte automobile multicompagnies.

Cliquer sur le lien ci contre    Flottes

3°) FICHE CLIENT : Clients Connexes

Dans votre portefeuille clients, certains d'entre eux ont des connexions.
exemple : le mari, la femme, l'entreprise du mari, l'entreprise de la femme.....
Vous devez donc savoir, en consultant la fiche du mari, qu'il existe des connexions avec les autres clients précités. Pour cela, la rubrique dossier dans la fiche client, vous permet de gérer ces connexions.  Cliquer sur le lien ci-contre : Clients connexes

4°) DOUBLE AFFICHAGE

Progressivement, nous supprimons le double affichage sur les écrans. Le bouton FRF aura tendance à disparaitre sur les écran.

5°) N° SS DANS LA LISTE DES ADHERENTS pour les contrats prévoyance

Lorsque vous imprimez la liste, une colonne Supplémentaire où figure le N° de sécurité social de l'adhérent

6°) CONTRATS : Les états

Une colonne supplémentaire a été insérée dans certains tableaux concernant les états contrats. Il s'agit de la colonne  Prime H. T.  Si vous voulez que cette information soit valide, cela suppose que dans le contrat, la prime HT (+ taxes) aient été renseignées.

7°) LOGICIEL E - CARE  Pour les utilisateurs du logiciel de sauvegarde FTP e-care, vous pouvez dorénavant sauvegarder des dossiers complets. Exemple, le dossier Mes documents.
