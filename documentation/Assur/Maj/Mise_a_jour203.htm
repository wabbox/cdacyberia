1) Evolutions

L'état de controle des quittances saisies est desormais imprimé en deux fois : une fois pour les quittances confiées, une fois pour les quittances non confiées.

Il comporte par ailleurs deux colonnes supplementaires : la sinistralité du client pour les deux années précédentes
2) Correction de problèmes signalés :

Lorsque vous faisiez l'import de votre terme, le logiciel tenait compte du premier apporteur pour calculer les rétrocessions, et pas du second.

Il tient compte désormais des deux apporteurs
