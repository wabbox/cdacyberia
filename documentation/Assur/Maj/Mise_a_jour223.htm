EVOLUTIONS

         1°) CONTRAT GROUPE ou CONTRAT COLLECTIF 
              

Il est possible maintenant d'envoyer un courrier à un adhérent ou à l'ensemble des adhérents

2°) SAISIE DES COMMISSIONS :  quittances Non confiées

Vous pouvez dorénavant dépointer les quittances non confiées en saisie des commissions. Ce "dépointage" fonctionne sur le même principe  que le rapprochement bancaire

3°) AGENTS MUTUELLE DU POITOU :  Conditions générales et EDI

Une rubrique supplémentaire a été ajoutée dans la fiche contrat afin de vous permettre de saisir la version des conditions générales.

Dans les EDI, Mutuelle du Poitou met parfois deux fichiers dans le même envoi ; cela crée une erreur sur le code enregistrement. Le programme a été modifié pour en tenir compte.
