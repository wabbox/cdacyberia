INFORMATIONS

Rappel : Toutes ces évolutions ne sont pas accessibles en Version 6

 

1°)  ACCES DISTANT par la 3 G

Si vous êtes en multiposte, vous pouvez vous connectez à votre serveur quelque soit l'endroit où vous êtes. Vous devez disposez d'un ordinateur portable et d'une carte 3G.
C'est également possible si vous avez une carte Edge, mais nous n'avons pas pu faire les tests.

2°)  GESTION DES COLLABORATEURS

2 nouveaux boutons ont été rajoutés :  MAILING pour faire un mailing à tous les collaborateurs
et PROFIL (dans la fiche collaborateur).  Le bouton PROFIL vous permettra de définir directement les options accessibles ou non au collaborateur.

3°)  COURRIERS : LISTE DES CONTRATS D'UN CLIENT

Vous pouvez créer un courrier pour un client, où figureront la liste de tous ses contrats en vigueur.
2 variables ont été créées pour insérer cette liste :

listecontrats :  la liste des contrats est donnée ligne par ligne avec N° contrat et branche

listecontratsh : la liste des contrats est donnée avec les N° des contrats qui se suivent.

Pour insérer la variable :  Insertion  Expression 4D  Cliquer sur Expression  Cliquer Modifier
saisir le nom de la variable  cliquer OK  cliquer OK.

  4°)  SAISIE QUITTANCE : CONTROLE EXISTENCE

Vous saisissez une quittance avec une date de début et une date de fin d'échéance.
Si vous saisissez une nouvelle quittance avec les mêmes dates d'échéance, le logiciel
lance une alerte  "une quittance pour la période du XXX au XXX existe déjà Confirmez vous ?"

4°)  GESTION PATRIMOINE : Droits de succession

Lorsque vous faites une simulation droits de succession, à partir de la fiche client, le détail du patrimoine est affiché.
