MODIFICATIONS ASSUR   MAJ  201 AU 17/10/2002

 
 

Page précédente          Accueil                     Les mises à jour

 

1) Evolutions :

          Nouveaux modules optionnels Debut septembre 2002, vous avez reçu un courrier pour vous informer des nouveaux modules optionnels :

a) Le profil social - fiscal et patrimonial du client

Les différents modules :  profil social - profil  fiscal - analyse patrimonial - reconstitution de carrière....

Vous avez pu constater, qu'il existe maintenant, sur la fiche client, le bouton PROFIL

b) Les E. D. I.  : Echanges de données informatisées avec les compagnies  Axa Courtage, Mutuelle du Poitou  et AGF Courtage.  L'EDI évite la double saisie (sur le système Cie et ensuite dans le logiciel ASSUR. Vous saisissez sur le système Cie. Cette dernière vous renvoie un fichier que vous intégrez dans ASSUR.. C'est le même principe que les importations des quittances terme.

Rappel : Coût de l'EDI :  230,00 Euros TTC  par  EDI en version monoposte et 280,00 Euros en version multipostes

Vous trouverez, dans la documentation sur Internet, partie Paramètres, utilitaires, les procédures pour intégrer les EDI de chaque compagnie.

Acompte : Dans la saisie ou le remboursement d'un acompte, vous avez maintenant la possibilité de créer un courrier.

Trésorerie : Préparation de la remise de chèques.  A l'affichage vous pouvez trier vos chèques soit par montant croissant ou décroissant, soit par nom. Vous pouvez également limiter la remise de chèques à 80 chèques.

Portefeuille Cabinet :  En compagnie, gestion des compagnies, le bouton portefeuille (hors contrat résiliés)  vous permet d'obtenir la composition de votre portefeuille global. Si vous désirez ce portefeuille, pour une compagnie, sélectionner la compagnie, modifier compagnie et cliquer sur portefeuille.

Comptabilité générale : Impression des comptes généraux.  La dernière colonne solde a été supprimée.  Le solde  apparait maintenant à la dernière ligne.

Règlement de sinistres : Un nouveau pop up menu vous permet de préciser le bénéficiaire du règlement. Si vous voulez rajouter d'autres bénéficiaires : en paramètres, gestion des énumérés, sélectionner l'énuméré s_beneficiaire et créer vos valeurs.

 Compagnies : Dans la fiche compagnie, des boutons complémentaires ont été créés mais à utiliser, uniquement si vous avez l'EDI.

Double affichage : si cette option est cochée dans les paramètres généraux,  lors de l'édition des relevés ou des avis d'échéance, le montant en Francs sera affiché

Consultation : 2 nouveaux critères ont été rajoutés : interlocuteurs et N° cie

Interlocuteur : Saisie dans la fiche client entreprise

N° Cie :  il s'agit du N° du client à la Compagnie.
