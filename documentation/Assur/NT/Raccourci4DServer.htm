Sommaire Notes Techniques e_Assur

Pour un lancement automatique du 4D Server, il est nécessaire de modifier les caractéristiques du raccourci.

1)Créer le raccourci sur le bureau

2) Aller dans les propriétés du raccourci :

 

3) Dans la zone Cible, rajouter, apres C:\ACI\Programs\4DServer.exe un espace, puis :

C:\Assur\Assur2V6.4DC

4) Dans la zone Démarrer en : effacer le contenu et remplacer par : C:\Assur\Assur2V6.4DC

Cliquer sur Appliquer, puis OK

Si vous désirez que la base soit lancée au demarrage, de l'ordinateur, Faites un Copier du raccourci puis allez dans :

Démarrer -------> Paramètres -------> Barre des tâches et Menu démarrer

Cliquez sur l'onglet Programmes du menu Démarrer, puis sur le bouton Avancé

Double-cliquez sur le répertoire Programmes, double-cliquez sur le répertoire Démarrage et faites un Coller. Le raccourci vers 4DServer.exe doit s'ajouter à la liste.
