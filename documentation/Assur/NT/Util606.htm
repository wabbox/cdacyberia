Utilisation de 4DUTIL 6.06

Page précédente

Preliminaires :

    Effectuer une copie de sauvegarde du fichier assur2v6.4DD

    Dans le dossier Poste de travail c: \ Maintenance Assur

Mono poste :

    Ouvrir le poste de travail, aller sur le disque c:

    Cliquer sur le répertoire Assur, puis sur le répertoire 4dutil606 et enfin double cliquer sur le programme 4Dutil.exe.

Multi postes :

    Ouvrir le poste de travail, aller sur votre disque c:

    Cliquer sur le répertoire ACI,  sur le répertoire Programs puis sur le répertoire 4Dutil606 et enfin double cliquer sur le programme 4Dutil.exe.

 Une fois le programme 4Dutil lancé, une fenêtre Ouvrir s'affiche, sélectionner alors le fichier assur2v6.4DC ( poste de travail \ c: \ Assur \ ) et entrer le mot de passe super_utilisateur.

Un écran gris s'affiche avec des menus.

Choisissez le menu Utilitaires qui propose 3 options :

  • Vérifier et réparer
  • Compacter
  • Tri permanent

 

Compacter :

4D Util va recopier le fichier de données Assur2V6.4DD en le compactant.

Si vous sélectionnez cette option, 4D Util vous présente une fenêtre d'enregistrement de fichier : Créer un fichier de données.Il propose de nommer le nouveau fichier Asur2V62.4DD et de l'enregistrer dans C:\Assur\.

Cliquez sur enregistrer.

Le compactage commence, son déroulement est décrit par une suite de messages.

 

 

Vérifier et Réparer :

 4D Util va recopier le fichier de données Assur2V6.4DD en le réparant :

    Dans la fenêtre suivante cocher : Récupération par analyse des Marqueurs de fiche et cliquer sur OK.  

    Dans la nouvelle fenêtre nommée " Créez un fichier de données "  la rubrique :  Nom du fichier doit inscrire automatiquement assur2v6_2.4DD. Cliquer  alors sur Enregistrer.

    Une fois la récupération terminée, quitter le programme 4Dutil.

Comparer :

    Il faut maintenant comparer la taille des deux fichiers assur2v6.4DD et assur2v6_2.4DD ( sélectionner le fichier en cliquant une fois puis bouton droit, rubrique propriété. La taille du fichier apparait dans cette fenêtre. Faire de même pour le second fichier )

    Les fichiers se trouvent dans : Poste de travail \ c: \ Assur

Vérifier

     Il faut également vérifier le nombre des fiches par éléments de assur2v6.4DD.

    Ouvrir le logiciel e-Assur dans le menu Utilitaire, cliquer sur " A propos de assur2 ", noter les résultats optenus et quitter le logiciel.

    Renommer ensuite le fichier assur2v6.4DD en Xassur2v6.4DD.

    Et renommer le fichier assur2v6_2.4DD en assur2v6.4DD.

    Ouvrir à nouveau le logiciel e-Assur dans le menu Utilitaire, cliquer sur " A propos de assur2 ", noter les résultats optenus et quitter le logiciel.

    Comparer les résultats pour obtenir le nombre de fiches perdues. Ces fiches ( ou écritures ) sont à retrouver et à ressaisir.

Attention :

    Le fichier de donnée que vous souhaitez conservez et utiliser doit toujours s'appeler assur2v6.4DD.

    Si vous ne voulez pas utiliser le nouveau fichier pensez à supprimer ou à déplacer le nouveau fichier assur2v6.4DD et renommer le fichier Xassur2v6.4DD en assur2v6.4DD.
