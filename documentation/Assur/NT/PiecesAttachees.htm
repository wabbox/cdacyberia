Lorsque vous consultez votre courrier électronique, certains messages contiennent une piece jointe. Vous les reconnaissez au fait qu'ils ont, à gauche un trombonne.

Cette pièce jointe est un fichier que vous devez enregistrer sur votre disque. Double-cliquez sur la ligne du message. Vous obtenez l'ecran de lecture du message. Dans la zone intitulée Joindre vous avez 1 ou plusieurs icones representant le ou les fichiers à télécharger.Dans l'exemple c-contre, il y en a 3.

Si vous désirez les télécharger toutes les trois, allez dans le menu Fichier, Enregistrer les Pièces jointes.
Le logiciel vous demande où vous voulez les enregistrer : dans l'exemple ci-contre, il vous propose de les enregistrer dans le répertoire Mes documents. Si cela vous convient, cliquez sur Enregistrer. Sinon, cliquez sur parcourir pour changer le répertoire de destination.
Si vous ne désirez télécharger que l'un des documents, par exemple 4DWrite.GID, double-cliquez sur son icone.Vous obtenez l'ecran suivant :
Vérifiez que c'est bien l'option "L'enregistrer sur le disque" qui est cochée, et cliquez sur OK

Le logiciel vous demande où vous voulez l'enregistrer . Dans l'exemple ci-contre, il vous propose de l'enregistrer dans le répertoire "Mes documents". si cela convient, cliquez sur Enregistrer. Sinon, cliquez sur le menu déroulant pour sélectionner le répertoire de destination.
