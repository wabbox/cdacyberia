CHANGEMENT D'ORDINATEUR
1) EN MONOPOSTE

Il suffit de transférer sur le nouvel ordinateur le répertoire où se trouve l'application, et de recréer le raccourci pointant sur l'application

En version 6 : Répertoire Assur Application : Assur2V6.EXE

En version 7 : Répertoire AssurV7 Application : Assur2V7.EXE

En version 8 : Répertoire AssurV8 Application : Assur2V8.EXE

En version 2007 : Répertoire Assur2007 Application : Assur2V9.EXE

Si le client est abonné à la sauvegarde FTP, il faut transférer en outre l'application de sauvegarde e_Care

Cette application se trouve dans un répertoire e_Care, lui meme placé dans le répertoire CAS, généralement placé dans le répertoire Program Files.

Transférer ce répertoire au meme endroit sur le nouveau matériel, et recréer le raccourci pointant sur l'application e_Care.EXE.

Cette application doit etre mise en démarrage automatique dans le menu Démarrer

Attention : si l'adresse de l'application Assur a changé, il faut modifier l'adresse des fichiers à sauvegarder dans le paramétrage de e_care

 

 

2 - EN MULTIPOSTE : CHANGEMENT DU POSTE SERVEUR EN VERSION 6

Il faut transférer : L'application 4D Server, le répertoire de l'application Assur, et transférer les droits

- répertoire de l'application 4D Server: ACI, généralement placé à la racine. Créer le raccourci sur 4DServer.EXE

- répertoire des droits : ACI, placé dans le répertoire système, (Windows ou Winnt)

- répertoire de l'application : Assur

Faire pointer le raccourci du 4DServer sur l'application Assur : Propriétés du raccourci, onglet Raccourci , rajouter le chemin l'application Assur2V6.4DC

Exemple : Cible : "C:\ACI\4D6O6\4D_Server_6O6\4DServer.exe" c:\Assur\Assur2V6.4DC

Si le client est abonné à la sauvegarde FTP, il faut transférer en outre l'application de sauvegarde e_Care

Cette application se trouve dans un répertoire e_Care, lui meme placé dans le répertoire CAS, généralement placé dans le répertoire Program Files.

Transférer ce répertoire au meme endroit sur le nouveau matériel, et recréer le raccourci pointant sur l'application e_Care.EXE.

Cette application doit etre mise en démarrage automatique dans le menu Démarrer

Attention : si l'adresse de l'application Assur a changé, il faut modifier l'adresse des fichiers à sauvegarder dans le paramétrage de e_care

 

 
3 - EN MULTIPOSTE : CHANGEMENT DU POSTE SERVEUR EN VERSION 7

Il faut transférer : L'application 4D Server, le répertoire de l'application Assur, et transférer les droits

- répertoire de l'application 4D Server: 4D674, généralement placé à la racine, ou dans program Files. Créer le raccourci sur 4DServer.EXE

- répertoire de l'application Assur : AssurV7

- transfert  des droits : sur l'ancien serveur, lancer le 4DServer. Aller dans le menu Fichier , Option Mise a jour des licences. Relever tous les numéros de série affichés, ou faire une copie d'écran.

Faire pointer le raccourci du 4DServer sur l'application Assur : Propriétés du raccourci, onglet Raccourci , rajouter le chemin l'application Assur2V6.4DC

Exemple : Cible : "C:\4D674\4D_Server_674\4DServer.exe" c:\Assur\Assur2V7.4DC

Lancer 4D server. Dans le menu Fichier Option Mise a jour des licences, saisir les numéros de série et les numéros d'expansion relevés sur l'ancien

Déclarer 4DServer.exe comme service : Menu Fichier, Option enregistrer comme service . Puis, dans les services, déclarer 4DServer en démarrage automatique

Si le client est abonné à la sauvegarde FTP, il faut transférer en outre l'application de sauvegarde e_Care

Cette application se trouve dans un répertoire e_Care, lui meme placé dans le répertoire CAS, généralement placé dans le répertoire Program Files.

Transférer ce répertoire au meme endroit sur le nouveau matériel, et recréer le raccourci pointant sur l'application e_Care.EXE.

5- CHANGEMENT DU POSTE CLIENT Version 6

1) Copier sur le nouveau client les deux  dossiers suivants :

- C:\Aci                    (de l'ancien client)

- C:\Windows\Aci   (de l'ancien client)

ATTENTION : LES REPERTOIRES COPIES DOIVENT AVOIR LE MEME CHEMIN D'ACCES.

Sur le nouveau Client, il doit donc y avoir :

  - 1 dossier à la racine du disque C:\  (Aci)                          - 1 dossier dans le répertoire Windows (Aci)

Vous recréez le raccourci sur 4DClient.exe

2) Sur le nouveau Client :

Supprimer les fichiers ASSUR2V6.REX et ASSUR2V6.RES

(Ces fichiers se trouvent dans C:\Windows\Aci)
