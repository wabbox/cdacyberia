PROCEDURE POUR FAIRE LES MISES A JOUR   e - Assur  SUR INTERNET
 

1ère étape : Téléchargement

Aller sur le site cda-informatique.fr.  http://www.cda-informatique.fr Vous arrivez sur la page d’accueil.  Cliquer sur mise à jour.

Vous êtes sur la page des téléchargements.
Cliquez sur la plateforme concernée (PC ou Mac)
Sur la nouvelle page, apparait la liste de toutes les versions Assur.

Les mises à jour se font pour les Monopostes, sur l'ordinateur où il y a le logiciel,  et pour les Multipostes : SUR LE SERVEUR.

 Cliquer sur celle qui vous concerne et ENREGISTRER LE FICHIER dans le dossier AssurXX   qui est sur votre
disque ou sur le bureau.  En général, le dossier AssurXX est sur le disque C:

Une fois le téléchargement terminé,  Cliquer sur Nouveautés  et imprimer le ou les n° de mises à jour qui vous
concerne.

Ensuite quittez Internet.

2ème étape : Décompactage

VERIFIEZ QUE VOUS AVEZ BIEN QUITTE :

-          Pour les versions monopostes : le logiciel  e - ASSUR

-          Pour les versions multipostes

o        Sur les postes clients : quitter le logiciel  e - ASSUR

o        Sur le poste serveur :  quitter 4 D serveur

Après avoir quitté Internet, aller sur le poste de travail, sur C :  et double cliquerez dans le dossier  AssurXX ( ex :assurV7 ou AssurV8) ou sur le bureau si vous avez enregistré sur le bureau.

Vous avez à l’intérieur le fichier que vous venez de  télécharger.

Double cliquez dessus

Un message en anglais – cliquer sur OK –  Un nouvel écran avec à gauche  la rubrique

C:\assurxXX Vérifiez que le décompactage se fait bien dans le bon dossier et sur le bon disque et cliquez sur unzip.

Un message vous informe que les fichiers ont été décompactés avec succès

 La mise à jour est faite.

3ème étape : Relancer le logiciel

 Pour les multipostes relancer le 4D serveur   Puis un poste client.  Lorsque vous arrivez sur le menu général,  les autres postes clients peuvent se connecter.

Pour les versions monopostes : relancer le logiciel

4ème étape : Vérification que la mise à jour s’est bien faite en contrôlant sur le menu general, au dessous de la rubrique sinistres, que le N° affiché correspond à la dernière mise à jour.
