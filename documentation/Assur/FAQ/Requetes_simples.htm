Requêtes simples

 

1) Rechercher les clients ayant un numéro de téléphone :

 

Table : Clients

Champs concernés : adresse1_Telephone1, adresse1_telephone2, adresse1_telephone3

Il faut rechercher les clients qui ont un telephone1, ou un telephone2, ou un telephone3

 

 

 

On veut imprimer la liste des clients triée dans l’ordre alphabétique, et les trois numéros de téléphone

On veut aussi avoir le nombre de ces clients

 

 

 

On ne veut pas avoir la liste des clients, mais seulement leur nombre ( on ne veut pas avoir le détail): sélectionner la ligne détail

et cliquer sur le bouton cacher (8ème à partir de la gauche)

 

 

2) Rechercher les clients n’ayant pas de numéro de téléphone :

 

Table : Clients

Champs concernés : adresse1_Telephone1, adresse1_telephone2, adresse1_telephone3

Il faut rechercher les clients qui ont ni telephone1, ni un telephone2, ni un telephone3

 

 

 

Les états seront les mêmes que précédemment.

 

3) Imprimer la liste des clients avec leur RIB

Tables concernées : Clients et RIB

a) Faire une requête sur la table clients (pour avoir la liste de tous les clients, demander les clients dont le numero est different de 0

 

b) puis construire l'état

cet état comporte des colonnes concernant deux tables : la table Clients et la table RIB

Vous créez vos colonnes. Pour que le logiciel puisse faire le lien entre les deux tables, vous créer une colonne supplémentaire dans laquelle il y aura une formule

Allez dans le menu Colonnes, et faites Ajouter. L'éditeur de formule vous invite à saisir la formule.

 

 

4) Recherche des véhicules non sortis d'une flotte

Il faut envisager deux cas

        a) Si cette flotte est gérée en branche Auto (code extension 4) la date de sortie figure dans le véhicule

        b) Si cette flotte est gérée en branche Flotte (code extension 27 pour les flottes multicompagnies) , la date de sortie se trouve dans la table Appartient_A_Flotte

 

a) Cas d'une flotte en branche Auto

On fait la requête sur la table Vehicules

Dans Liste des champs, sélectionner : Toutes les tables

 

 

 

a) Cas d'une flotte en branche Flotte auto

On fait la requête sur la table Vehicules

Dans Liste des champs, sélectionner : Toutes les tables
