Les fichiers de mise a jour pour le module Gestion de patrimoine sont les suivants  :
  • Impôts (IR, ISF, Droits de succession)
  • Cotisations sociales
  • Paramétrage des Etats
  • Paramétrage des plans de gestion
  • Types de produits
  • Produits
  • Tables de mortalité
  • Paramétrage des analyses
  • Types de biens

Pour télécharger un fichier : cliquez sur le lien correspondant avec le bouton droit de la souris, et sélectionnez l'option : "Enregistrer la cible sous ...."

Enregistrez le fichier dans le répertoire EDI de votre disque dur.
