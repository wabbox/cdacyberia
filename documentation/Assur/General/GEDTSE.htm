Sommaire Production

 
Lorsque vous travaillez en accès distant, l'utilisation de la GED est un peu plus compliquée car votre application est lancée sur le serveur, à des kilomètres de chez vous,et non sur votre ordinateur en local. Par contre le scanner est relié à votre ordinateur local, et les fichiers à stocker dans les dossiers clients sont sur votre ordinateur local.

Vous devez donc, avant de travailler , paramétrer le serveur et le client, afin que le serveur puisse lire des fichiers sur le disque du client.

 

  
 1 - Paramétrage du serveur

Le serveur doit voir le disque du client :

Aller en démarrage --> Outils d'administration --> Configuration des services Terminal Server

Cliquer sur Connexions Cliquez droit sur RDP-Tcp

-->Option Propriétés --> Onglet Paramètres du client

L'option Mappage des lecteurs doit être décochée.

Dans le répertoire de l'application du client ASP, créer un répertoire GedStockCodeClient
2 - Sur le poste du client distant

Dans le paramétrage de la connexion RDP Connexion bureau à distance, Onglet ressources locales, Périphériques locaux,  cocher Lecteurs de disque.

Créer sur le disque dur un répertoire GedScan .
Dans e_Assur : Paramètres --> paramètres généraux, paramétrer les chemins d'accès.

  
3 - Intégrer un document ou une image

Lorsque vous etes sur l'écran Ged du client ou du contrat...... cliquez sur Charger document. Le dialogue d'ouverture de document s'affiche

 
Remontez dans la hiérarchie des dossiers jusqu'à ce que vous arriviez au poste de travail. Vous voyez une liste de disques dont un au moins est indiqué comme partagé, le votre (dans l'exemple ci-contre C sur JC, D sur JC ).
Double cliquez sur votre disque local , et allez localiser le document que vous voulez intégrer dans le dossier du client.

Dans l'exemple ci-contre, il s'agit de "Domogest_macro_declaration"

Cliquez sur Ouvrir
Le document s'ouvre dans l' application d'origine. Fermez le, et cliquez sur le bouton Stocker.
4 - Scanner un document

Vous ne pouvez pas scanner directement à partir de l'application. Donc, dans un premier temps, vous scannez avec l'application de scannage fournie avec le scanner.

Enregistrez le document scanné dans le répertoire Gedscan de votre disque dur. Puis vous procédez comme en 3) pour intégrer le document dans le dossier du client.
