Codes Postaux
Sommaire
Vous disposez d'une base de données de 40000 codes postaux. Vous avez toutefois la possibilité d'ajouter, de modifier ou de supprimer des codes postaux.

La première rubrique permet de saisir tout ou partie du code postal :

exemple : 69@ vous donnera la liste de toutes les communes du département 69

Pour Ajouter un code postal : cliquer sur ajouter

Pour Modifier un code postal (ou la ville) sélectionner le dans la liste et cliquer sur Modifier.

Pour Supprimer un code postal ou une ville : sélectionner le dans la liste et cliquer sur Supprimer
