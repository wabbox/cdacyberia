Liste des programmes d'importation de Terme disponibles

Pour installer un programme d'importation de terme, vous devez procéder en trois étapes : Récupérer le fichier d'installation nommé EDI_XXX.TXT  (XXX représente le nom de la compagnie. Par exemple :EDI_GENERALI.TXT ) et procéder à son installation. Vous devez ensuite vérifier que l'installation s'est bien faite. Enfin vous devez vérifier le paramétrage afin qu'il corresponde à vos paramètres personnels.

1°)Première étape : Récupérer le fichier d'installation 

Allez dans la documentation e_assur, dans la partie paramètres, et cliquez sur le lien Liste des termes disponibles. La liste des programmes que vous pouvez télécharger est affichée. Cliquez avec le bouton droit de la souris . Sélectionnez l'option Enregistrer la cible sous ... et enregistrez le fichier dans le répertoire EDI de votre disque s'il existe, sinon dans tout répertoire de votre choix.

Si le programme dont vous avez besoin ne figure pas dans la liste, CDA vous l'aura envoyé en pièce jointe d'un e_mail

Vous devrez alors récupérer dans votre courrier électronique le fichier que CDA vous a adressé en pièce jointe, et l'enregistrer sur le disque dans le répertoire de votre choix. Ce fichier s'appelle EDI_XXX.TXT

Pour cette opération, voir : Enregistrer des pièces attachées

Lorsque le fichier est sur le disque, dans le répertoire de votre choix, allez en Gestion ----->Compagnies -------> Gestion des compagnies

Sélectionnez la compagnie concernée, cliquez sur Modifier.

Sur l'écran de la compagnie, cliquez sur Importer EDI

Le logiciel vous demande de localiser le fichier EDI_XXX.TXT :

 
 
2) Deuxième étape : Contrôler l'installation du programme

 

  • Contrôlez que le programme a bien créé sur votre disque dur un répertoire EDI dans lequel se trouve un fichier EDI_XXX 
  • Controlez que tous les programmes nécessaires à l'EDI sont bien installés. Pour cela, allez en Paramètres --> Utilitaire --> Importations de fichiers

Vérifiez que vous avez bien dans la zone de défilement de droite la ligne:

XXX Quittances a bien été créée.

Par exemple Generali Quittances ou Rhodia Quittances, etc...
  3) Adapter le programme à votre configuration propre.

 En particulier, vérifiez que le code de la compagnie concernée correspond bien au code de votre compagnie. Pour cela, sélectionnez la ligne correspondant au terme que vous venez de récupérer, cliquez sur Modifier cliquez sur le menu déroulant Compagnie, et sélectionnez la compagnie concernée. Validez.
