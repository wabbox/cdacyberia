L'EXPORTATION D'ETATS ET DOCUMENTS

Documentation Generale produits CDA

Sommaire Documentation e_Assur

 

Il est possible, pour certains documents,  de les émettre non pas par impression mais par création de fichier, afin de pouvoir les transmettre à leur destinataire sous forme de fichier, en pièce jointe d'un e_mail. (envois de fonds, bordereaux de rétrocession, états de quittances ....) . Par exemple, lorsque vous êtes en envois de fonds, vous pouvez imprimer le bordereau par les liens Tester Bordereau ou Imprimer Bordereau. Vous pouvez aussi créer un fichier et l'envoyer par e_mail à la compagnie, à l'aide du lien Envoyer Bordereau.

L'écran de gestion des exports se présente de la manière suivante :

- A droite, les descriptifs disponibles. Si vous n'avez pas celui qui vous convient, vous pouvez le télécharger sur notre site ou le créer vous-mêmes après avoir suivi la formation adéquate. Pour télécharger, cliquez sur Télécharger. (Voir la Liste des descriptifs possibles)

Lorsque vous avez enregistré le descriptif qui vous intéresse dans votre répertoire EDI, vous cliquez Importer. (Voir Installation d'un descriptif d'export)

- A droite, 4 cases à cocher vous permettant de donner la composition du document :
  • Entête comporte des informations concernant l'expéditeur et le destinataire
  • Titre des colonnes créera un titre pour chaque colonne
  • Corps du document : ce sont les différentes lignes du document (par exemple la liste des quittances composant l'envoi de fonds)
  • Totaux : la dernière ligne comportant les totaux des colonnes numériques (prime, commission, etc ..sera transmise au fichier

- A gauche , les modalités de création et de transmission du fichier :
  • Trois boutons radio pour sélectionner le format d'export( TXT, XML, Excel, en fonction des indications données par le destinataire( la compagnie par exemple )
  • Le répertoire dans lequel sera stocké le fichier produit (Pour faciliter la maintenance, nous vous conseillons d'utiliser le répertoire C:\EDI)
  • Le nom sous lequel le fichier créé sera stocké
  • La case à cocher Envoyer à l'adresse : Si vous la cocher, le fichier sera envoyé en pièce jointe à l'adresse indiquée.

Dans ce cas, l'écran se modifie pour vous permettre de saisir l'objet et le texte du message.

Dans certains cas, vous pouvez aussi exporter sous Excel les états standard e_Assur

Par exemple : les statistiques de sinistralité.

Pour cela, cliquez sur Etat Standard. La liste des etats s'affiche à droite. Si dans la liste des etats disponibles il y en a un dont le nom se termine par Excel, c'est que vous pouvez exporter cet état vers Excel. Voir l'exemple Statistiques de sinistralité.
