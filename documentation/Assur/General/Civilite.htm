I - SIGNATURE AUTOMATIQUE D'UN COLLABORATEUR

Il est désormais possible de faire signer automatiquement un courrier par un collaborateur.

Pour cela :

1 - En gestion des collaborateurs : saisir vos collaborateurs
dans la rubrique  signature  courriers :  saisir la signature
cocher la rubrique :  Accès à la base.

2 - En gestion des énumérés, dans l'énuméré m_sign, mettre en valeur la liste des collaborateurs qui peuvent signer un courrier.

3 - Dans les courriers, paramétrer la zone qui doit recevoir le nom du collaborateur de la manière suivante :

Insertion -  Expression  4D -  Cliquer sur l'onglet Expression
Cliquer modifier  Saisir sur le nouvel écran  v_sign  cliquer sur OK  puis OK.
La variable v_sign est affichée (pour la voir affichée   Edition - Afficher les références)

4 - Lors de l'émission du courrier, choisir dans le menu déroulant, rubrique signature, le collaborateur concerné.

II - DEFINITION D'UNE CIVILITE

Pour définir la civilité dans un courrier, 3 procédures sont disponibles :

aqui                                             aqui2                                   aqui3

La procédure aqui permet de mettre devant le titre du client "Cher" ou "Chère"

On paramétrera donc de la façon suivante : Menu base - option Insérer une commande 4D.

Une fenêtre "Nom de la variable ou de la commande à insérer" apparaît à l'écran.

Saisir : aqui et cliquer sur OK. Vous obtenez "aqui"

Choisir dans le menu base l'option : insérer un champ. Sélectionner le fichier clients. Cliquer sur la rubrique titres. Cliquer sur OK. Vous obtenez : (clients)TITRE

La procédure aqui2 permet d'inscrire en civilité :

"Cher Client" si le titre est Monsieur ou Maître

"Chère Cliente" si le titre est Madame ou Mademoiselle

"Chers Clients" dans tous les autres cas.

On paramètrera : Menu base - Option : Insérer une commande 4D.

Une fenêtre "Nom de la variable ou de la commande à insérer" apparaît à l'écran.

Saisir : aqui2 et cliquer OK. Vous obtenez : "aqui2"

La procédure aqui3 permet d'inscrire en civilité :

"Cher Monsieur" ou "Cher Maître" si le titre est Monsieur ou Maître

"Chère Madame" ou "Chère Mademoiselle si le titre est Madame ou Mademoiselle

"Messieurs" dans tous les autres cas.

On paramètrera : Menu base - option : insérer une commande 4D.

Une fenêtre "Nom de la variable ou de la commande à insérer" apparaît à l'écran.

Saisir : aqui3 et cliquer sur OK. Vous obtenez "aqui3"
