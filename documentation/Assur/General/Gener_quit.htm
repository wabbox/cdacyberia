Sommaire Production
L'objet de cette option est de générer les quittances non confiées ou confiées pour éviter de les saisir.

Exemple :

Sur un contrat maladie, le client règle au cabinet une première quittance comptant couvrant deux ou trois mois. La quittance comptant est saisie en confié, puis, les mois suivants, les règlements se font par prélèvement automatique. Afin de pouvoir contrôler les bordereaux compagnie, il est nécessaire de disposer des quittances termes mensuelles correspondant aux prélèvements mensuels de la compagnie.

Chaque mois vous devez :

- Générer toutes les quittances mensuelles non confiées.

- En saisie des commissions pointer les quittances qui font l'objet du bordereau compagnie

 

La gestion automatique des quittances se fait uniquement sur les informations saisies dans la police. Vérifiez donc que les informations : jour et mois d'échéance, echéance de début, échéance de fin, prime, commission, fractionnement sont correctes

POUR GENERER VOS QUITTANCES

1- Vous précisez le mois sur lequel les quittances doivent être générées : 1 pour janvier, 2 pour février, etc....

2 - Vous précisez la date à laquelle ces quittances doivent être enregistréees et l'exercice. par exemple 1/1/2004 exercice 2004

3 - Sélectionnez :

- la compagnie et éventuellement la branche (cochez dans ce cas une compagnie) ou cochez une entité si vous désirez éditer les quittances sur toutes les compagnies

- le fractionnement si vous ne désirez que les quittances d'un certain fractionnement (par exemple uniquement les quittances mensuelles)

- Indiquez si vous souhaitez des quittances confiées ou non (coché = encaissement confié)

Cliquez sur Validation
Lorsque les quittances ont été générées, un message vous informe du résultat de l'opération.

Allez en Quittances, Etat de contrôle, pour imprimer la liste des quittances générées.
