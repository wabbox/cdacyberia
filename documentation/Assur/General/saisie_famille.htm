SAISIR LA FAMILLE D'UN CLIENT
Cette fonction vous permet de saisir les informations concernant la famille du client : enfants, parents, conjoint etc...

L'objectif est triple : avoir une bonne connaissance du client, pouvoir exploiter ces données de manière commerciale, utiliser ces informations pour les simulations dans le module Gestion de Patrimoine.

La case à cocher est importante pour l'édition des attestations scolaires : la variable EnfantsClients donne la liste de tous les enfants du client, alors que la variable EnfantsScolarisés ne donne que les enfants dont la boite à cocher Scolarisé est cochée.
