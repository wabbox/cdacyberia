Sommaire Production

Il est possible de repérer les clients connexes en créant un dossier dans lequel seront regroupes tous les clients ayant un une connexion entre eux. Par exemple l'entreprise Demo, le chef d'entreprise en tant que particulier, une autre entreprise dont il est le gérant, etc..

Vous pouvez mettre dans un dossier autant de clients que vous le désirez.

Pour cela, il suffit de choisir un code qui sera le numéro de dossier

 

Sur la fiche client, de tous les clients d'un même dossier, vous inscrivez le numéro de dossier commun, par exemple ici : Durand

Lorsque vous consulterez un client faisant partie d'un dossier, la zone correspondant au numéro de dossier sera surlignée en jaune, et le numéro et le nom d'un autre membre du dossier sera affiché sur le menu déroulant à droite

 

 
Si vous désirez voir l'ensemble des membres du dossier, cliquez sur la flèche du menu déroulant  
Vous retrouverez cette information d'appartenance à un dossier sur différents écrans : contrat, compte client, etc.... avec toujours la signalétique jaune
