Sommaire Paramètres

Pour installer l'EDI GAN EUROCOURTAGE, vous devez procéder en deux étapes :

1) Récupérer le fichier d'installation nommé EDI_GAN.TXT et procéder à son installation

(2) Procéder au contrôle du paramétrage des programmes EDI

1°)Première étape : Récupérer le fichier d'installation 

Vérifiez que vous avez bien sur votre disque un répertoire EDI. Si ce n'est pas le cas, créez le.

Si, dans ce répertoire, vous n'avez pas de répertoire EDI_GAN, créez le.

Puis :

 Allez en Gestion ----->Compagnies -------> Gestion des compagnies

Double-cliquez sur la compagnie GAN

Sur l'écran de la compagnie, cliquez sur le bouton EDI

Sur le nouvel ecran, cliquez sur le bouton Télécharger EDI

 
La page contenant les liens vers les EDI téléchargeables s'affiche
Cliquez droit sur la ligne GAN Eurocourtage, et sélectionnez l'option Enregistrer la cible sous....Le dialogue d'enregistrement de fichier vous demande où enregistrer de fichier EDI_GAN.TXT

 

Localisez le répertoire EDI_GAN dans EDI, et cliquez sur Enregistrer

Quittez votre Navigateur web.

Cliquez sur le bouton Installer EDI
Le dialogue d'ouverture de fichier vous demande où se trouve le fichier EDI_GAN.TXT à installer. Allez de sélectionner dans le répertoire EDI_GAN et cliquez sur Ouvrir
Le logiciel vous informe lorsque l'installation est terminée
2) Controle du paramétrage du programme EDI

Controlez que touts les programmes nécessaires à l'EDI sont bien installés. Pour cela, allez en Parametres --> Utilitaire --> Importations de fichiers

Vérifiez que vous avez bien dans la zone de défilement de droite les lignes :

  • EDI GAN AFN Auto
  • EDI GAN AFN GAV
  • EDI GAN AFN Habitation
  • EDI GAN AFN reference immeuble
  • EDI GAN reference professionnels
  • EDI GAN reference professions liberales
  • EDI GAN avenant changement adresse
  • EDI GAN avenant changement de syndic
  • EDI GAN avenant changement immat
  • EDI GAN Entete production
  • EDI GAN Entete terme
  • EDI GAN Production Debut
  • EDI GAN reglement sinistre
  • EDI GAN Sinistre
  • EDI GAN Sinistre debut
  • EDI GAN Terme

plus, si vous avez installé le message 502 les deux lignes :

  • Gan quittances FCA export Entete
  • Gan quittances FCA export

3) Controle du paramétrage général

En paramètres généraux; vous devez avoir saisi votre identifiant FCA

Sur la fiche de la compagnie GAN, à la page EDI, vous devez avoir saisi : la version du message 502 AA, le code FCA de la Compagnie
