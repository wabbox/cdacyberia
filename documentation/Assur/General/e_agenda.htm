Sommaire Documentation e_Assur

Sommaire Paramètres
e_Agenda vous permet de gérer vos rendez-vous et ceux de vos collaborateurs.

Vous pouvez l'utiliser en hébergement sur le site de CDA Informatique, ou intégré à votre application e_Assur.

Dans les deux cas vous pouvez l'utiliser en local , sur vos ordinateurs, ou à distance à l'aide de tout équipement relié à internet : ordinateur de bureau, ordinateur portable équipé d'un modem GPRS, Wifi, assistant personnel, téléphone mobile.

LE PARAMETRAGE

 

1 - L'hébergement

Sur le site de CDA, vous et vos collaborateurs êtes identifiés par le code de votre entité, votre code collaborateur, et un mot de passe.

Vous nous demandez l'ouverture d'un compte, et vous créez autant de collaborateurs que vous le désirez.. Si vous vous connectez à e_Assur en tant que super_utilisateur, créez un collaborateur super_utilisateur.

Ces collaborateurs doivent être créés dans votre logiciel e_Assur , (en gestion des collaborateurs), avec acccès à la base. En gestion des profils utilisateurs, vous leur donnez l'accès à l'option Agenda Rendez-vous. Puis vous les créez sur e_Agenda.

En paramètres généraux, cliquez sur le bouton accès

Dans la zone URL de l'agenda, saisissez : www.cda-informatique.com

Dans votre navigateur, intégrez aux favoris l'adresse : www.cda-informatique.com/votreentité/codecollaborateur

 

2 - e_Agenda intégré à e_Assur monoposte

Téléchargez les deux fichiers de paramètres à l'adresse :

http://www.cda-informatique.fr/logiciels/e_agendaMedia.exe

http://www.cda-informatique.fr/logiciels/e_agendaPages.exe

Double-cliquez sur chacun de ces deux fichiers pour les décompacter dans le répertoire AssurV8.

Les collaborateurs ayant accès à l'agenda doivent être créés dans votre logiciel e_Assur , (en gestion des collaborateurs), avec acccès à la base. En gestion des profils utilisateurs, vous leur donnez l'accès à l'option Agenda Rendez-vous.Si vous vous connectez à e_Assur en tant que super_utilisateur, créez un collaborateur super_utilisateur.

En paramètres généraux, cliquez sur le bouton accès

Dans la zone URL de l'agenda, saisissez : 127.0.0.1

Dans votre navigateur, intégrez aux favoris l'adresse publique de votre ordinateur, par exemple 193.254.56.12/votreentite/votrecodecollaborateur

 

 

3 - e_Agenda intégré à e_Assur multiposte

 

Téléchargez les deux fichiers de paramètres à l'adresse :

http://www.cda-informatique.fr/logiciels/e_agendaMedia.exe

http://www.cda-informatique.fr/logiciels/e_agendaPages.exe

Double-cliquez sur chacun de ces deux fichiers pour les décompacter dans le répertoire AssurV8.

 

Les collaborateurs ayant accès à l'agenda doivent être créés dans votre logiciel e_Assur , (en gestion des collaborateurs), avec acccès à la base. En gestion des profils utilisateurs, vous leur donnez l'accès à l'option Agenda Rendez-vous.Si vous vous connectez à e_Assur en tant que super_utilisateur, créez un collaborateur super_utilisateur.

En paramètres généraux, cliquez sur le bouton accès

Dans la zone URL de l'agenda, saisissez l'adresse privée de votre serveur , par exemple : 192.168.1.2

Dans votre navigateur, intégrez aux favoris l'adresse publique de votre serveur ou de votre routeur, par exemple 193.254.56.12/votreentite/votrecodecollaborateur

 

LES ADRESSES IP

Pour connaître l'adresse IP privée d'un ordinateur sur un réseau, aller en démarrage -->Programmes -->accessoires -->invite de commande

taper : IPCONFIG et valider

Pour connaître votre adresse publique : connectez vous à l'adresse http://www.cda-informatique.com
