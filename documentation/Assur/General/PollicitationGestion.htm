Lorsque vous sélectionnez en Paramètres -->Courriers l'option Pollicitation, deux cas peuvent se présenter :

- Vous n'avez pas encore paramétré de pollicitation, dans ce cas, vous arrivez sur l'écran de création d'une pollicitation

- Vous avez déjà paramétré une pollicitation, dans ce cas, vous arrivez sur l'écran de gestion des pollicitations.

 
- Les boutons Modifier et Supprimer vous permettent d'aller sur l'écran de création/modification d'une pollicitation
- Le bouton Recalculer vous permet d'affecter à postériori une pollicitation à des quittances déjà saisies
- Le bouton Formule sur contrats vous permet d'interdire ou d'autoriser la pollicitation sur une sélection de contrats.
- Le bouton Formule sur clients vous permet d'interdire ou d'autoriser la pollicitation sur une sélection de clients.
- Les boutons Contrats pollicités ou Contrats quittancés vous permettent d'obtenir la liste des contrats qui ont fait l'objet d'une pollicitation ou qui, après une première souscription, font l'objet d'un quittancement.
 

LES CAS OU LE LOGICIEL NE POLLICITERA PAS UNE QUITTANCE

Lorsque vous saisissez ou importez une quittance, le logiciel peut refuser de lui associer une pollicitation :

 

1) Il vérifie tout d'abord que le client ou le contrat n'est pas interdit de pollicitation

2)Il recherche ensuite les pollicitation disponibles sur les critères : code branche, code sous branche, entité

3) S'il s'agit d'une entreprise, il vérifie si les entreprises ne sont pas exclues sur les pollicitations trouvées

4) S'il s'agit d'un particulier, il vérifie si les entreprises ne sont pas exclues sur les pollicitations trouvées

5) Il vérifie si ce client ou ce contrat ayant déjà été pollicité, n'a pas refusé la pollicitation

6) Si c'est une pollicitation client, il vérifie qu"un autre contrat de ce client n'a pas déjà été pollicité
