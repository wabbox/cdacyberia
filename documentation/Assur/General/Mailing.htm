Option : Paramètres              Menu : Courriers                   Option : Mailing

Envoyer un mailing à une sélection de clients se fait en deux temps :

1) Faire une requête pour sélectionner les clients auxquels le message doit être envoyé

2) sélectionner le support : courrier en traitement de texte, mail en format texte, mail en format html mis en page sous 4D Write, mail en format html mis en page à l'aide d'un editeur HTML

 

 

I- Les requêtes

Voir quelques exemples de requêtes :

Exemple de requête avec plusieurs tables

Le résultat de votre requête est affiché sous forme d'une liste de clients . Si vous désirez exclure des clients de l'envoi, vous cochez la case à cocher à gauche.

 
II - Le choix du type de support

1) Le courrier : Si vous désirez imprimer un courrier, vous cliquez sur le bouton Courrier

Vous sélectionnez le courrier à imprimer, puis vous cliquez sur validation.
2) Le mail : Si vous désirez envoyer un e_mail, vous cliquez sur le bouton E_Mail

Pour evoyer un texte tout simple, vous saisissez votre texte et vous cliquez sur e_mailing

Pour envoyer un courrier traitement de texte transformé en HTML,  cliquez sur Html .
Le logiciel vous propose de sélectionner le courrier à envoyer. Sélectionnez le et cliquez sur validation. Le texte 4D Write est affiché. Vous pouvez éventuellement le modifier avant envoi.

Puis cliquez sur e_mailing.
Pour envoyer un document Html créé à l'aide d'un éditeur, cliquez sur Html . Ne sélectionnez aucun courrier, et cliquez sur Validation. Sur la page d'envoi du mail, cliquez sur le bouton HTML externe. Localisez le document HTML qui a été créé au préalable

Puis cliquez sur e_mailing
