PROCEDURE POUR CREER VOS CARTES VERTES EN TRAITEMENT DE TEXTES

Il est possible d'imprimer des cartes vertes directement à partir du logiciel.
Les formats étant différents pour chaque compagnie, c'est à l'utilisateur de créer, autant de modèle de cartes vertes qu'il a de compagnies.

Quelques modèles sont à votre disposition :

  1. Pour les intégrer dans Assur,  cliquez droit Enregistrer la cible sous.
    Enregistrer sur le bureau.  Puis dans Assur  - Dans courrier  courriers   Nouveau - donner le nom de votre carte verte.  Fichier ouvrir  et sélectionnez le fichier qui est sur le bureau. Validez votre courrier.  Puis en gestion des compagnies, sélectionnez la compagnie et dans rubrique format carte verte, sélectionnez le format concerné.
  2. Modèles de Carte vertes

II - CREATION

En courrier - courrier, Cliquer sur ajouter et créer un modèle de cartes vertes.

Donner un nom à ce courrier, exemple cverteaxa

Paramétrer le courrier avec les rubriques suivantes :

Pour les valeurs saisies :  Menu Insertion    Expression 4D   cliquer onglet  CHamp   et double cliquez sur Extension puis
sélectionner le champ désiré

[extension]Veh_immat                           n° d'immatriculation

[Contrat]numérocontrat                         n° de police

[extension]categorie                                 catégorie du véhicule

[extension]marque                                    marque du véhicule

Pour les autres informations figurant sur la carte verte   MENU    BASE   INSERER UNE COMMANDE 4D et saisir les variables suivantes : 

(ou en version 7 :  Menu Insertion   - Expression 4D  - Cliquer sur l'onglet expression  et saisir les variables suivantes, une à une

da1                                                        date de début de validité

da2                                                        date de fin de validité

Adresse client : insérer les variables suivantes

  • v_adr_cli_1   Titre Nom Prénom
    v_adr_cli_2   Adresse sur 1
    v_adr_cli_3   Adresse sur 2
    v_adr_cli_4   Code postal  et ville
    v_adr_cli_5   Pays
    v_adr_cli_6   Représentant
    v_adr_cli_7   Enseigne

Cliquer ensuite sur  (  Affichage  sélectionner l'option références) pour vérifier que les valeurs ci-dessus ont bien été paramétrées.

Valider ensuite votre modèle.

Puis par le principe du Copier coller, créer vos autres modèles de cartes vertes

Pour chaque modèle de carte verte que vous créez, lors de l'ajout du nouveau courrier, vous saisissez le nom et dans la rubrique FONCTION : sélectionner Cartes Verte.

ATTENTION, LE NOM QUE VOUS AVEZ DONNE A VOTRE "COURRIER" DEVRA ETRE RENSEIGNE EXACTEMENT DANS LA GESTION DES COMPAGNIES, DANS LA CASE FORMAT DE CARTES VERTES.

Il faut donc en Compagnie, gestion des compagnies,  sélectionner la compagnie concernée par la carte verte, cliquer sur Modifier et en bas à droite, dans la rubrique  format carte verte,  selectionner  votre courrier  cliquer validation

 Maintenant, c'est à vous de jouer pour la mise en page.
