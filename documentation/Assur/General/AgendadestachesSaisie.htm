Sommaire Agenda
L'agenda des taches permet de programmer des taches pour l'avenir.

Il existe deux sortes de taches : les taches que vous programmez vous meme, et les taches automatiques programmées par le logiciel.

Les taches programmées par le logiciel : cliquez sur le lien
Les taches saisies : vous pouvez programmer des taches des deux manières :

Soit à partir de l'écran Contacts des fiches Client, Contrat, Sinistre, soit par l'option du menu Agenda : Saisir Agenda

 
- La date : la date de début est celle à laquelle l'alerte doit etre affichée, la date de fin n'est là qu'à titre d'information si cette taches doit s'effectuer sur plusieurs jours

- L'émetteur et le destinataire :si vous programmez une tache pour vous meme, l'émetteur et le destinataire sont les memes, sinon précisez qui est le destinataire

- client, contrat, siniste, extension : ces informations sont remplies automatiquement si vous saisissez à partir de la fiche contact, sinon vous pouvez les saisir.

- priorité : vous pouvez indiquer le niveau de priorité de cette tache

- si vous cochez tache programmée, l'alerte correspondante sortira indéfiniment au rythme que vous précisez

- a droite, la liste des courriers types vous permet de programmer l'envoi d'un courrier à  une date précise. Lorsque vous sélectionnez un courrier, deux boutons radio apparaissent vous permettant d'indiquer si vous voulez imprimer le courrier ou l'envoyer par mail. Si vous avez des taches repetitives, vous pouvez vous créer une liste déroulante des menus déroulants

- L'objet : c'est un résumé explicite de la tache, qui sera affiché dans les listes de taches

- La zone de texte vous permet d'expliciter le contenu de la tache

 

- le bouton Messagerie vous permet d'envoyer un message instantané à un autre utilisateur. Pour pouvoir utiliser cette fonctionnalité, vous devez avoir effectué au préalable deux paramétrages :

                         - En parametres --> Paramètres généraux   cochez la case Messagerie qui se trouve dans la partie Interface

                         - sur le 4D Server, Menu Edition, Option Preferences, Dans Application  -->Client-serveur, cochez Inscrire les clients au démarrage

- le bouton Rendez-vous vous permet d'accéder à l'agenda Rendez-vous
