4D WRITE est le logiciel de traitement de textes intégré à la base de données.

Il utilise le principe de base d'un traitement de textes.

De ce fait, nous considérons que les notions standard manipulées dans tout logiciel de traitement de textes vous sont familières (saisie et édition, marges,tabulations, etc...) ainsi que les commandes copier,coller, effacer, couper...

Vous allez concevoir vos courriers types. Les courriers types livrés en standard sont   totalement modifiables.

Un courrier est composé :

- de texte, saisi au clavier et formaté selon les besoins de l'utilisateur (gras, italique, centré...)

- de rubriques (ou champ) que l'on insère dans le texte   (exemple : les coordonnées du client, le numéro de contrat...)

- d'informations calculées, soit des variables.

Exemple : Dans l'avis d'échéance, vous voulez voir apparaître,

Sur 1 ligne                         la prime TTC                             Rubrique saisie dans la quittance

Sur 1 seconde                   les frais de facturation                  L'information figure également dans la quittance

Sur une troisième ligne      le total de ces 2 sommes             Rubrique calculée

Le total est donc une information calculée. Il faudra donc utiliser le Menu Insertion, option Expression 4D, cliquer sur l'onglet Expression, cliquer sur le bouton Modifier,  et saisir le nom de la variable. Le nom de cette variable vous est donné.

Si vous voulez un calcul spécifique, nous appeler.

INSERTION D'UN CHAMP

Toutes les saisies faites dans e_ASSUR peuvent être insérées dans le texte.

Exemple : Vous voulez insérer le Nom du client.

Dans le Menu Insertion, sélectionner l'option Expression 4D,

L'écran qui est affiché vous donne la liste des champs de la table Client

Double cliquez sur Nom
