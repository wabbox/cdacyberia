Sommaire Documentation e_Assur

Sommaire Paramètres

Paramètres ----->Paramètres ------->Départements

Lorsque Vous sélectionnez cette option, e_Assur affiche la liste des départements : Le nom et le numéro. Les deux colonnes suivantes concernent la tarification santé : elles indiquent si le département bénéficie du tiers payant pour le régime général SS et pour le régime TNS.

Si vous désirez modifier les caractéristiques d'un département, double-cliquez sur sa ligne.
