Sommaire Paramètres

Paramètres --->Utilitaires --->EDI AGF

1) Pour la procédure d'installation de l'EDI, cliquez ici

2°)Première étape : Récupérer les fichiers AGF 

Vous devez tout d'abord récupérer dans votre courrier électronique les fichiers qu'AGF vous a adressés en pièces jointes, et les placer dans votre répertoire de travail.

Par défaut, ce répertoire est : "C:\EDI\EDI_AGF"

Ces fichiers sont nommés :

  • AUTO.TXT pour la production auto

  • IRD.TXT pour la production IRD

  • SIN.TXT pour les mouvements sinistres

  • TERME_ENTR.TXT pour les quittances des contrats entreprise

  • TERME_PART.TXT pour les quittances 

Pour cette opération, voir : Enregistrer des pièces attachées

 

3)Deuxième étape : Executer le programme EDI AGF

En Paramètres, Utilitaires, sélectionnez l'option EDI AGF

Vous obtenez l'écran suivant :

Dans la zone de défilement de gauche, vous avez la liste des importations que vous avez déjà réalisées. En haut et à droite, les boîtes à cocher vous permettent de sélectionner les opérations que vous voulez importer.

Il n'y a pas dans l'EDI AGF d'informations concernant les règlements de sinistres. Cette option est donc inactive.

Vous n'avez pas à localiser le fichier à utiliser : les fichiers AGF portent toujours le même nom.

En ce qui concerne les quittances, seul le mois de comptabilisation est indiqué par la compagnie. Vous devez donc préciser la date de comptabilisation , l'exercice, et  le mois.

4) Contrôle

Lorsque l'EDI est terminée, vous devez vérifier ce qui s'est passé. Vous sélectionnez l'EDI du jour dans la zone de défilement du jour, puis vous cliquez sur Voir

Vous obtenez l'écran de compte-rendu :

 

Dans la colonne fonction, le type de la fonction : client - contrat - quittances - sinistres....

Dans la colonne opération, les opérations effectuées : création,  modification , ......

La colonne Anomalie vous permettra de diagnostiquer, rapidement le problème rencontré lors de cette importation. En particulier, le message "A vérifier : création de client sur nom " signifie que le logiciel n'a pas trouvé le client relatif à la création d'un contrat. Lorsqu'il ne le trouve pas par le numero compagnie, il le cherche sur le nom, le début du prénom, et le code postal. S'il ne le trouve pas, il le crée. Il est donc important chaque fois que vous aurez ce message, de vérifier si la création est justifiée ou non.

Vous pouvez imprimer cette liste des opérations importées.

Au fur et à mesure des importations, vous aurez dans la colonne de gauche, Importation, une liste de toutes vos importations.  Pour conserver,  à l'affichage,  uniquement les dernières opérations, sélectionner celles qui doivent disparaitre et cliquer sur  EPURER .

Le bouton DETRUIRE : NE JAMAIS UTILISER CE BOUTON SAUF sur indication de CDA.(Il supprime les opérations comptables relatives aux quittances).
