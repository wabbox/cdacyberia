Sommaire Paramètres

On trouve des adresses dans deux endroits différents :

- sur le client, on peut saisir une adresse 1 et une adresse 2
-  sur chaque contrat, on peut saisir une adresse de querabilité

LES ADRESSES DANS LE CLIENT

Vous pouvez saisir une seule adresse pour le client. Dans ce cas, tous les courriers seront envoyés à cette adresse.
Vous pouvez saisir 2 adresses pour le client. Dans ce cas, vous pouvez préciser la seconde adresse, en cliquant sur la bande 2eme adresse et en cochant "c'est l'adresse du client" et préciser les dates.

Vous pouvez donc vous trouver dans 3 situations :

- Vous saisissez 2 adresses mais vous ne cochez pas la boite à cocher. Les courriers seront envoyés à l'adresse 1.
- Vous saisissez 2 adresses,vous cochez la boite à cocher, mais vous ne saisissez pas de dates. Les courriers seront toujours envoyés à l'adresse 2.
-Vous cochez la boite à cocher, et vous mettez une date de début et unedate de fin : les courriers seront envoyés à l'adresse 2 dans l'intervalle de dates, et à l'adresse 1 en dehors de cet intervalle.

LES ADRESSES DANS LES CONTRATS - SINISTRES - QUITTANCES

Dans chaque contrat, vous pouvez saisir une adresse de querabilité, avec des dates de validité.Cette adresse de querabilité sera utilisée pour les courriers envoyés à partir d'un contrat, d'une quittance ou d'un sinistre.

S'il n'y a pas d'adresse de querabilité, ou si ladate du jour est en dehors de l'intervalle de date, e-assur ira chercher l'adresse dans la fiche client, selon les principes définis ci-dessus, pour les adresses clients.

 

LE PARAMETRAGE DES COURRIERS

Pour pouvoir utiliser cette gestion des adresses, vous ne devez pas utiliser les champs de la base de données puisque ceux ci peuvent être différents.

Vous devez donc paramétrer des variables calculées par le logiciel. Ce paramétrage se fait en gestion des courriers.

En version  6 : Menu Base ---> Insérer une commande 4D
En version è : Menu Insertion ---> Expression 4D ---> cliquer sur onglet  Expression

Saisissez les variables suivantes  pour l'adresse du client

v_adr_cli_1   Titre Nom Prénom
v_adr_cli_2   Adresse sur 1
v_adr_cli_3   Adresse sur 2
v_adr_cli_4   Code postal  et ville
v_adr_cli_5   Pays
v_adr_cli_6   Représentant
v_adr_cli_7   Enseigne
