INSTALLATION DES EDI AXA COURTAGE

Documentation Generale produits

Sommaire Documentation e_Assur

Sommaire Paramètres

Pour installer l'EDI AXA, vous devez procéder en trois étapes :

1) Récupérer le fichier d'installation nommé EDI_AXA.TXT et procéder à son installation

2) Procéder au paramétrage des branches et des garanties

3) Procéder au contrôle du paramétrage des programmes EDI
1°)Première étape : Récupérer le fichier d'installation 

Vérifiez que vous avez bien sur votre disque un répertoire EDI. Si ce n'est pas le cas, créez le.

Si, dans ce répertoire, vous n'avez pas de répertoire EDI_AXA, créez le.

Puis :

 Allez en Gestion ----->Compagnies -------> Gestion des compagnies

Double-cliquez sur la compagnie AXA

Sur l'écran de la compagnie, cliquez sur le bouton EDI

Sur le nouvel ecran, cliquez sur le bouton Télécharger EDI

 

 
La page contenant les liens vers les EDI téléchargeables s'affiche
Cliquez droit sur la ligne AXA Courtage, et sélectionnez l'option Enregistrer la cible sous....Le dialogue d'enregistrement de fichier vous demande où enregistrer de fichier EDI_AXA.TXT

Localisez le répertoire EDI_AXA dans EDI, et cliquez sur Enregistrer

Quittez votre Navigateur web.

Cliquez sur le bouton Installer EDI
Le dialogue d'ouverture de fichier vous demande où se trouve le fichier EDI_AXA.TXT à installer. Allez de sélectionner dans le répertoire EDI_AXA et cliquez sur Ouvrir
Le logiciel vous informe lorsque l'installation est terminée
2) Deuxième étape : Controler, corriger, compléter les paramétrages branches et garanties

Pour que l'EDI puisse fonctionner correctement, il est nécessaire de donner à e_Assur un certain nombre d'informations.
Paramétrage des branches et sous branches :

Les contrats créés doivent être rattachés à vos branches et sous branches

Les messages AXA Courtages donnent deux informations sur la classification en branche et sous branche : 

1) Le type de contrat : AUP, HAB, RCP

2) Le code UV du produit :03254,25227 ...... pour l'auto, 03714, 03724 ... pour l'habitation

 

Un partie du paramétrage a été faite, mais vous devez la compléter.

Pour cela, allez en Gestion ---->Compagnies----->Gestion des compagnies

Sélectionnez la compagnie AXA Courtage, cliquez sur Modifier. Sur l'écran de la compagnie, cliquez sur Branches

Vous obtenez le tableau qui établit la correspondance entre les branches AXA Courtage et les vôtres

 

Si, sur une des lignes, vous voyez un (?), c'est que e_assur n'a pas su trouver la branche ou la sous branche. A vous de corriger.
Pour modifier une ligne existante, double-cliquez sur la ligne correspondante, vous obtenez l'écran de saisie, et vous modifiez la branche et la sous branche.

Pour rajouter une ligne, double-cliquez sur une ligne blanche
3) Controle du parametrage du programme EDI

 

  • Controlez que le programme a bien créé sur votre disque dur un répertoire EDI dans lequel se trouve un repertoire EDI_AXA ou EDI_AXA Courtage
  • Controlez que touts les programmes nécessaires à l'EDI sont bien installés. Pour cela, allez en Parametres --> Utilitaire --> Importations de fichiers

Vérifiez que vous avez bien dans la zone de defilement de droite les lignes :

  • Axapac Entete
  • Axapac MVT Sinistre
  • Axapac production
  • Axapac quittance de prime
  • Axapac Reglement sinistre
