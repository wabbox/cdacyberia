Sommaire courriers
Souvent, les informations que vous avez besoin d'imprimer dans un courrier ne peuvent etre obtenues par insertion d'un champ( soit parce que l'information n'est pas stockée dans la base, soit parceque e_assur a besoin de calculer cette information)

Dans ce cas, vous devez insérer une expression, et non un champ.

Pour cela, il faut dans Menu Insertion   Expression 4D   Cliquer sur l'onglet Expression , cliquer sur Modifier, sur le nouvel ecran saisir dans le rectangle blanc le nom de la variable ou de l'expression et cliquer sur Validation
  1 - Informations se rattachant au client

L'adresse du client peut etre paramétrée à l'aide des champs de la table clients. Mais dans ce cas, le logiciel ne peut pas tenir compte d'une éventuelle seconde adresse, ou d'une adresse de quérabilité.

Il vaut donc mieux paramétrer une variable :

Vous pouvez paramétrer l'adresse en un seul bloc :

v_adr_cli

ou vous pouvez paramétrer chaque ligne séparément :

v_adr_cli_1, v_adr_cli_2, v_adr_cli_3, v_adr_cli_4, v_adr_cli_5,v_adr_cli_6, v_adr_cli_7

Les dates d'exercice de l'entreprise :

courrier_v_datedebut_exercice,courrier_ v_datefin_exercice, courrier_v_periodeexercice

La liste de ses contrats :

courrier_v_Listecontrats

La liste de ses enfants :

courrier_v_Liste_Enfants, courrier_v_Enfantsscolarisés

Le nombre d'enfants :

courrier_v_Nbenfants, courrier_v_nbenfants_ACharge

 

2 - Informations se rattachant au contrat

Les quittances dues :

courrier_v_quittancesdues

Le total du :

courrier_v_quittances_total

La liste des ayant droit :

courrier_v_Listeayantdroit, courrier_v_listeayantdroitSante courrier_v_ListeayantdroitSE (sante enfants), courrier_v_listeayantdroitP (prevoyance), courrier_v_listeayantdroitAss (assistance)
 

La liste des garanties :

courrier_v_listegar, courrier_v_listegarsous (souscrites), courrier_v_listegarliste

La liste des enfants de l'adhérent :

courrier_v_Listeenfadherent

 

Coordonnées de l'apporteur

AppCode, AppTitre, Appnom, Appprenom, AppAdresse1, AppAdresse2, AppCodePost, AppVille, Apppays, Apptel,  Appfax

Coordonnées du gestionnaire

Gestcode, gesttitre,  Gestnom, Gestprenom, GestAdresse1, GestAdresse2, Gestcodepost, Gestville, Gestpays, Gesttel, Gestfax

3 - La variable d'adresse générique

Vous pouvez etre amené à envoyer un courrier à une adresse autre que celle du client, par exemple à partir d'un sinistre vous envoyez un courrier à un interlocuteur dans une compagnie, ou chez un expert , .....

Dans ce cas, il vaut mieux paramétrer une adresse non spécialisée, valide quel que soit le destinataire :

v_adr_destinataire
