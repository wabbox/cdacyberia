Sommaire Comptabilité

FONCTION   : Comptabilité     - OPTION : Compta Gle        -      SOUS-OPTION : Opérations diverses
Cette option vous permet de saisir des opérations diverses en une écriture.

Lorsque vous la sélectionnez, vous obtenez l'écran de saisie :
Vous sélectionnez l'entité, vous indiquez la date de l'écriture, éventuellement vous modifiez l'exercice, vous saisissez le libellé correspondant à la première ligne, puis vous saisissez les lignes d'écritures.

Dans la colonne Numéro de compte, vous saisissez le numéro du compte. Si le compte existe, le libellé est affiché automatiquement.

S'il n'existe pas dans la comptabilité, un message vous en avertit, vous pouvez alors aller en gestion du plan comptable pour le créer.

Si ce compte est un compte centralisateur, vous devez saisir le divisionnaire : le code de la compagnie, le numéro du client, le code de l'apporteur, etc...

Vous saisissez ensuite le montant, soit dans la colonne débit, soit dans la colonne crédit.

.

Vous saisissez ainsi toutes les lignes de l'écriture,  puis vous cliquez sur validation.

Si l'écriture n'est pas équilibrée, un message vous en avertit, et la validation est refusée.

Si tout est en ordre, les comptes et le journal sont mis à jour.

Vous revenez ensuite sur l'écran de saisie.

Si vous n'avez pas d'autres opération à saisir, vous cliquez sur quitter.

Vous êtes alors de retour à l'écran principal.
