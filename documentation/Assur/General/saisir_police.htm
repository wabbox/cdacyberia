Vous pouvez saisir des fiches polices nouvelles pour un nouveau client, ou pour un client existant déjà.
1 - CAS DU NOUVEAU CLIENT

Le logiciel cherche si ce numéro existe déjà.

S'il n'existe pas, vous en êtes prévenu par un dialogue.

Vous cliquez OK, et la fiche de saisie du client apparaît.

Pour effectuer cette saisie, reportez vous au manuel : Gestion- Client - Saisir.

Lorsque vous cliquez sur VALIDATION, la fiche Police est affichée.

Pour la saisie de la fiche, voir ci-dessous en 4.

2 - CAS D'UN CLIENT EXISTANT

Tapez le numéro du client. Le logiciel cherche si le numéro existe déjà. S'il n'existe pas, vous êtes prévenu par un dialogue. Vous avez vraisemblablement fait une erreur de saisie du numéro.

Cliquez sur Modifier, et vérifiez le numéro que vous avez tapé.

S'il existe, les NOM et PRENOM sont affichés.

3 - SAISIE D'UNE POLICE

La date d'effet  est à saisir, ainsi que le n° de contrat. Vous pouvez également saisir le n° de client de la Compagnie.

Pour saisir la compagnie auprès de laquelle le contrat est souscrit, sélectionner, dans le menu déroulant, la compagnie désirée.

Si ce contrat doit être affecté à un apporteur, vous cliquez sur la zone  "producteur". La liste de vos apporteurs est affichée, vous sélectionnez celui qui est concerné.

S'il s'agit d'un contrat en coassurance, vous cliquez sur la boite à cocher "Coassurance".

Vous devez alors indiquer s'il s'agit du contrat de la compagnie apéritrice, et le pourcentage de participation.

Pour saisir les garanties, vous cliquez sur le bouton "Extension ou Garantie".

Pour saisir les garanties souscrites, vous sélectionnez la garantie dans la zone de défilement de droite, et vous saisissez éventuellement le montant de la garantie et la franchise. Elle apparaît dans la colonne de gauche.

Lorsque le saisie est terminée, vous cliquez sur "Validation Retour".

Par contre, si vous désirez saisir le détail, vous pouvez saisir les véhicules, les habitations, ou les adhérents.

Vous cliquez sur le bouton Validation et suivant, si vous avez d'autres risques à saisir.

 SAISIE DES GARANTIES : dans la zone de défilement de droite, vous avez la liste des garanties qu'il est possible de souscrire. Pour affecter une garantie à ce véhicule, sélectionner cette garantie, puis cliquez sur AJOUTER. La garantie souscrite est affichée dans la zone de défilement de gauche. Pour supprimer une garantie, sélectionnez-la dans la zone de défilement de gauche, et cliquez sur Corbeille. Lorsque toutes les garanties ont été saisies, cliquez sur VALIDATION. Vous pouvez alors saisir un nouveau véhicule. Lorsque tous les véhicules ont été saisis, vous cliquez sur QUITTER.

Double cliquer sur la garantie à garder et cliquer sur l'icône représentant une feuille déchirée (en dessous de la croix rouge), un message apparaît : Etes vous certain, cliquer OK

Blocage affaire pour pièces  manquantes : Vous pouvez créer des listes de pièces manquantes et dans chaque Menu déroulant, indiquer le Motif du Blocage de l'affaire.

Avenant : Vous visualisez la liste des Avenants, la date et le Motif.

Vous revenez alors à l'écran de saisie résumée des garanties. Vous pouvez soit revenir à la première page, soit valider à partir de cet écran.

Si le client a versé un acompte, vous cliquez sur "Acompte".

Vous obtenez l'écran de saisie d'un acompte.

Voyez le chapitre correspondant.

Si le client n'a pas versé d'acompte, cliquez sur "Validation"et l'écran de saisie d'une nouvelle police est affiché. Vous pouvez soit saisir une nouvelle fiche, soit cliquez sur Quitter si vous désirez cesser la saisie.

4 - Saisie quittance

Cette option vous permet de saisir immédiatement la quittance au comptant.  (voir option "Saisir quittance").

Historique : Cet écran vous permet de saisir plusieurs types d'information si vous le désirez.

Historique du contrat

- Proposition signée le  : Date à laquelle la proposition a été signée au client

- Affaire saisie le : Date à laquelle le contrat a été saisi par le commercial

- Affaire bloquée jusqu'au : Si des pièces manquent pour transmettre le contrat à la Cie

- Pièces manquantes reçues le : Si une proposition était bloquée pour pièces manquantes, on peut noter la date à   laquelle ces pièces ont été reçues.

- Prochain règlement terme : Date qui est dans le contrat est reportée.

- L'action commerciale de réalisation : Si le contrat a été souscrit à la suite d'une action commerciale dont vous désirez l'efficacité, vous devez indiquer à quelle action il doit être rattaché.
