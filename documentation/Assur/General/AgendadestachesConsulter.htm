Sommaire Agenda
Lorsque vous vous connectez, votre agenda des taches est affiché automatiquement

Vous pouvez par ailleurs, à tout moment aller consulter votre agenda des taches dans le menu  Agenda, Option Consulter Agenda
La liste des taches non effectuées est affichée à gauche .Pour voir le détail d'une tache, cliquez sur la ligne correspond
ante, elle apparaitra à droite. Pour modifier une tache, double-cliquez sur la ligne correspondante.

Pour indiquez qu'une tache a été éxécutée, cochez la case Exécuté le et indiquez la date d'éxécution..

Le bouton Imprimer un permet d'imprimer la tache sélectionnée.

Le bouton Imprimer Tout permet d'imprimer toutes les taches affichées .

Le bouton Ajouter vous permet d'ajouter une tache

Le bouton Détruire permet de détruire  la tache sélectionnée, le bouton Détruire tout permet de détruire toutes les taches affichées

Le bouton Effectué tout permet de marquer les taches affichées comme effectuées.

Le bouton Modifier permet de modifier la tache sélectionnée

Le bouton Reporter permet de reporter une tache à une date ultérieure.

Le bouton Courriers et mails permet de lancer l'impression des courriers et l'envoi des mails qui avaient été programmés pour ce jour

Le bouton Rendez-vous permet d'accéder à l'agenda rendez-vous

Vous pouvez demander à voir les taches d'un autre utilisateur, les taches sur une période donnée
