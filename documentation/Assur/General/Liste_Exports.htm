En bas de la plupart des listes, affichées par le logiciel, vous disposez du bouton "Envoyer vers"

Vous avez le choix entre envoyer votre liste vers une imprimante(Option Impression), ou de créer un fichier externe en format texte, XML, Excel (option Fichier externe)

Si vous disposez d'un logiciel de création de PDF, vous pouvez utiliser l'option Impression pour générer un document PDF que vous pourrez ensuite envoyer en pièce jointe.

Attention : les graphes ne peuvent pas utiliser l'option d'envoi automatique de mail.

 

 

1) L'option Impression :

(pour imprimer le document ou créer une document PDF)

a) Etat standard : Il s'agit d'imprimer un état qui a dejà été programmé dans le logiciel .Si vous cliquez sur le bouton-radio Etat standard , une liste déroulante apparaît à droite, donnant la liste des formats disponibles en standard.. Vous sélectionnez le format qui vous convient, et vous cliquez sur Executer
b) Etat Libre : Si aucun état standard ne vous convient, vous pouvez créer votre propre état vous-même, grâce à l'éditeur d'états semi automatiques . Cliquez sur le bouton-radio Etat libre, puis sur Executer

Pout télécharger le manuel PDF qui vous permettra de contruire vos propres états, cliquez ici
c) Graphe : Vous pouvez vouloir créer des graphes a partir de données chiffrées stockées dans la base.. Cliquez sur le bouton-radio Graphe, puis sur Executer

Pour télécharger le manuel PDF qui vous permettra de construire vos graphes,cliquez ici
d) Etiquettes : Si vous désirez envoyer un mailing aux personnes en lien avec cette liste, vous pouvez imprimer des étiquettes.Cliquez sur le bouton-radio Etiquettes, puis sur Executer

Pour savoir comment construire vos étiquettes,cliquez ici

Pour télécharger le manuel PDF qui vous permettra de construire vos étiquettes,cliquez ici
e) Mailing : Si vous désirez envoyer un mailing aux personnes en lien avec cette liste, vous pouvez imprimer un courrier Cliquez sur le bouton-radio Mailing, puis sur Executer. Le logiciel vous demande de sélectionner dans la liste des modèles-types le courrier que vous désirez envoyer.Sélectionnez le et cliquez sur Validation

Voir la documentation Mailing
e) e_Mailing : Si vous désirez envoyer un e_mailing aux personnes en lien avec cette liste,  cliquez sur le bouton-radio e_Mailing, puis sur Executer. Vous pouvez leur envoyer un e_mail en format texte : saisissez votre texte, puis cliquez sur le bouton e_mailing . Si vous désirez envoyer un e_mail en format HYML, cliquez sur le bouton HTML , et sélectionnez le modèle type que vous désirez envoyer et cliquez sur Validation, puis sur e_mailing

Voir la documentation concernant l'envoi d'e_mail

2) L'option créer un fichier externe

lI est possible, pour certains documents,  de les émettre non pas par impression mais par création de fichier, afin de pouvoir les transmettre à leur destinataire sous forme de fichier, en pièce jointe d'un e_mail. (envois de fonds, bordereaux de rétrocession, états de quittances ....) . Par exemple, lorsque vous êtes en envois de fonds, vous pouvez imprimer le bordereau par les boutons Tester Bordereau ou Imprimer Bordereau. Vous pouvez aussi créer un fichier et l'envoyer par e_mail à la compagnie, à l'aide du bouton Envoyer Bordereau.
Par exemple, vous désirez envoyer une liste vers Excel :Cliquez sur le bouton Excel

Le logiciel vous demande de préciser sous quel nom et dans quel répertoire vous voulez enregistrer le classeur Excel

Il vous demande aussi de quelle manière vous allez sélectionner les données à exporter.

Vous disposez de trois possibilités :

 
- Utiliser un descriptif : CDA a préparé un certain nombre d'exports spécifiques, par exemple pour les envois de fonds ou les états de portefeuille .La liste apparait à droite.Si le descriptif existe, vous le sélectionnez. Puis vous indiquez la structure de la feuille que vous désirez créer :

- A droite, 4 cases à cocher vous permettant de donner la composition du document :

  • Entête comporte des informations concernant l'expéditeur et le destinataire
  • Titre des colonnes créera un titre pour chaque colonne
  • Corps du document : ce sont les différentes lignes du document (par exemple la liste des quittances composant l'envoi de fonds)
  • Totaux : la dernière ligne comportant les totaux des colonnes numériques (prime, commission, etc ..sera transmise au fichier

Si votre document n'apparait pas dans la liste de droite, vous pouvez le télécharger , ou le créer vous meme après avoir suivi la formation adéquate. Pour télécharger, cliquez sur Télécharger. (Voir la Liste des descriptifs possibles)

Lorsque vous avez enregistré le descriptif qui vous intéresse dans votre répertoire EDI, vous cliquez Importer. (Voir Installation d'un descriptif d'export)

 

- Choisir les champs : Vous pouvez sélectionner vous meme les champs à exporter . Cliquez sur Choisir  les Champs et cliquez sur Executer

Pour télécharger le manuel PDF qui vous permettra de construire vos exports,cliquez ici
3) Envoyer le fichier

Vous pouvez envoyer le fichier ainsi créé à un de vos correspondants : cochez la boite à cocher Envoyer à l'adresse

Vous saisissez l'adresse du correspondant, ou vous indiquez le type de correspondant(Cie, collaborateur, contact), et vous le selectionnez dans le menu déroulant correspondant. Vous saisissez l'objet du message, éventuellement le texte du message, et vous cliquez sur Executer. Lorsque l'operation d'export(impression ou création d'un fichier a été effectuéee, le logiciel vous demande si vous voulez envoyer le fichier. Vérifiez que vous avez bien quitté l'application qui a créé le fichier, puis cliquez sur le bouton Envoyer.
