Sommaire Documentation e_Assur

Sommaire Paramètres
Cette option vous permet d'alléger votre base de données des enregistrements non utiles.(comptabilité d'années antérieures, courriers anciens, contrats résiliés ...

Attention, cette opération peut prendre un temps certain.

L'option Epurer vous permet de supprimer physiquement les enregistrements de la base sans possibilité de récupération. 

L'option Archiver vous permet de supprimer physiquement les enregistrements de la base et de créer des fichiers externes à sauvegarder afin de pouvoir, si nécessaire, récupérer les enregistrements supprimés.

L'option Récupérer vous permet de réintroduire dans la base les enregistrements archivés précédemment.
Marquer les clients comme supprimés : Un certain nombre de clients n'ont pas de contrats, ou ont tout leurs contrats résiliés depuis un certain temps.

Le logiciel inscrira dans la fiche client une date de suppression. Cela aura pour effet de faire disparaître ces clients de certaines listes , par exemple des mailings. Par ailleurs, si vous demandez l'épuration des clients, ceux-ci seront épurés.

Clients : Supprime les clients marqués supprimés antérieurement à la date indiquée, et tout ce qui leur est rattaché : les historiques comptables, les contrats, les quittances , les sinistres, les courriers. Attention donc à ne pas faire de suppressions sur la période de reprise fiscale !

Comptes clients lettrés : Supprime les historiques comptables clients qui ont été lettrés avant la date indiquée.

Polices résiliées : Supprime les contrats résiliés avant la date indiquée, et tout ce qui s'y rapporte : quittances sinistres, extensions, habitations, courriers .

Courriers : Supprime tous les courriers émis avant la date indiquée.

Comptabilité : Supprime Grand Livre et balance pour l'exercice indiqué. Attention donc à ne pas faire de suppressions sur la période de reprise fiscale !
Tous les fichiers d'archivage ont été placés dans un répertoire Archives, dans votre répertoire de travail. Vous devez sauvegarder ce répertoire Archives.

Lorsque toutes les suppressions demandées ont été réalisées, de nombreux trous ont été faits dans la base de données et il faut procéder à un compactage afin de bénéficier de la place ainsi libérée . Pour cela, vous devrez utiliser 4D Util ou 4D Tools.
Récupérer : si vous avez besoin de récupérer des données qui ont été archivées, cochez Récupérer. Le logiciel vous demande de localiser le répertoire dans lequel se trouvent les données à récupérer.

Localisez le répertoire Archives sur votre disque.
La liste des fichiers à récupérer apparaît dans la zone de défilement.

Si vous cliquez sur Archiver , tous les fichiers marqués à gauche d'un > seront réimportés.

Si vous désirez exclure certains fichiers de la procedure de récupération, cliquez sur la ligne correspondante afin de faire disparaître le signe >.
