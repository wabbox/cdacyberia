Il existe 2 types de modèles d'avis d'echéance :

* Le modèle entièrement paramétré par l'utilisateur en traitement de textes

Vous pouvez télécharger des modèles en traitement de texte

- Pour la V8

          - avis d'échéance

* Le modèle composé d'une partie de texte, paramétré par l'utilisateur" et d'une partie de format prédéfini par CDA

 

Ci-dessous, les exemple de modèles prédéfinis

avisecheance :   Traitement de Textes frais de quittancement ligne séparée

avisecheance2 : Format 4D + Traitement de Texte + Frais de Quittancement ligne séparée

avisecheance3 : Format 4D + Traitement de Texte intégrant Frais de Quittancement

avisecheance4 : Format 4D uniquement intégrant frais de quittancement

avisecheance5 : Format 4D + texte non formaté + frais de quittancement
