Sommaire Utilitaires

 
- L'option  recherche dans une table vous permet de faire des requêtes spécifiques.

Certaines requêtes existent déjà en automatique :

- Clients : imprimer la liste des clients
- Contrats :  Requêtes courantes
- Sinistres : Requêtes courantes

Pour les autres requêtes,  utiliser la fonction  en Utilitaires - Recherches sur une table

Vous arrivez sur un écran et vous surlignez à gauche, la table concernée.
Vous arrivez sur l'éditeur de recherches - Vous faites votre recherche. La liste est affichée.
Cliquez envoyer vers : Cocher Etats standards et Executer pour imprimer la liste ou
Cocher Excel - Donner un nom a votre fichier et Executer
La table d'export apparait.  Sélectionner la table concernée
Puis créer vos colonnes, en fonction de vos besoins.
N'oubliez pas dans l'onglet  en tete  de cocher  Titre des colonnes.
Une fois le tableau créé, cliquez sur Exporter.


Par exemple, exporter la liste des clients avec leur RIB sous Excel
Utilitaires - Recherche sur une table On clique sur RIB puis Validation
L'éditeur de recherche est affiché
Dans notre cas,  Tiers_Numéro est égal à @ (pour avoir tous les clients)  puis cliquez rechercher.
Une liste est affichée. Cliquez sur Envoyer  Vers   Nouvel écran  cocher  Excel, donner un nom à votre fichier dans la rubrique sous le nom  RIBClients puis Executer

La table d'export apparait.  Sélectionner la table concernée, dans notre exemple, RIB.
  Pour les infos concernant le client, cliquez sur le signe + de la rubrique Tiers_ID
Créer votre tableau - onglet en tête  titre des colonnes  et cliquez exporter
Excel se lance et vous obtenez votre liste sous Excel.  Il suffira de l'enregistrer.
