Sommaire Gestion
Pour les contrats dont l'encaissement est confié, les bordereaux se font à partir des quittances .

Pour les contrats dont l'encaissement est non confié, les bordereaux se font à partir des contrats.

Dans le premier cas, utilisez l'option Bordereau d'envoi de fonds.

Dans le second cas, utilisez l'option Bordereau sur Polices.

 
1) La sélection

Vous pouvez imprimer ou envoyer par e_mail  ou envoyer un fichier que vous aurez généré vos bordereaux d'envois de fonds correspondant aux opérations d'une période donnée pour un client (sélectionnez le client), pour une compagnie(sélectionnez la dans la liste de gauche), pour toutes les compagnies(sélectionnez l'entité), pour un produit (sélectionnez le produit).

Vous définissez le contenu de ces envois de fonds : quittances terme, quittances comptant, acomptes (ceux qui ont été affectés à une compagnie), sinistres.
Vous indiquez si vous désirez produire un bordereau pour les quittances et un pour les ristournes , ou si vous désirez un seul bordereau pour quittances et ristournes.

Vous pouvez demander au logiciel d'émettre ces bordereaux automatiquement (Editer bordereaux), ou vous pouvez les visualiser à l'écran afin de vérifier et éventuellement en définir le contenu (préparer bordereaux).Si vous désirez envoyer vos bordereaux par e_mail, vous devez impérativement choisir Préparer bordereaux.

Les dates de sélection : vous pouvez sélectionner les quittances à échéance durant une période donnée qui ont été encaissées (sélection sur dates d'échéancee), ou les quittances qui ont été réglées pendant une période donnée (sur date d'encaissement).

Le bouton Définir Formats vous permet de sélectionner le format prédéfini du bordereau à imprimer

Vous disposez de 3 modèles d'envoi de fonds. Ci-dessous les 3 modèles à votre disposition

  • Modèle 1 : Envfonds

  • Modèle 2 : Envfonds2

  • Modèle 4 : EnvondsEAC

Pour paramétrer le modèle qui vous convient

 
 

2) La préparation : Le logiciel vous affiche la liste des quittances et/ou ristournes correspondant à votre sélection.

En haut de l'écran, sont affichés le total de l'envoi, et le solde de la compagnie.

Vous pouvez décider ne ne pas faire d'envoi de fonds pour une ou plusieurs quittances en cochant la boite à cocher correspondante.

Le bouton Tester bordereau vous permet d'imprimer le bordereau sans le valider (pour contrôle)

Le bouton Valider bordereau vous permet d'imprimer et de valider le bordereau

Le bouton Envoyer bordereau vous permet d'envoyer ce bordereau par e_mail à la compagnie, ou de générer un fichier qui sera ensuite transmis à la compagnie

 
2) L'envoi du bordereau

Pour envoyer un bordereau d'envois de fonds à la compagnie, vous devez avoir au préalable :

- saisi l'adresse e_mail à laquelle l'envoyer dans la rubrique e_mail règlement de la fiche compagnie

- saisi le code compagnie FCA (dans la fiche compagnie)

- saisi votre code courtier (dans la fiche compagnie)

- saisi la version du message envoyé (AA ou BB selon la compagnie)

- saisi votre code courtier FCA si vous en avez un (en paramètres généraux, Modifier Entité)

- importé le descriptif du fichier en Utilitaires, Importation de fichiers

 
Lorsque vous cliquez sur Envoyer bordereau, vous arrivez sur l'écran d'export de données

Cliquez Texte dans Fichier externe
Le logiciel vous demande comment vous désirez définir les champs à exporter. Cliquez sur Utiliser un descriptif
Sélectionnez dans le tableau de droite le descriptif correspondant à la compagnie. Vérifiez que entête et corps du document sont cochés.

Comment télécharger un descriptif

Comment installer un descriptif

 
Cochez Envoyer à l'adresse : l'adresse e_mail du destinataire et l'objet du message sont affichés. Corrigez si nécessaire.

 

Cliquez sur Executer.

Le logiciel génère le fichier, puis vous demande de confirmer l'envoi. Cliquez sur Envoyer

L'e_mail avec sa piece jointe est adressé au destinataire

Le logiciel vous demande ensuite si vous désirez valider le bordereau . Si tout s'est bien déroulé, confirmez la validation.
