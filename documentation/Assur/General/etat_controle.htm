Sommaire Quittances
Lorsque vous saisissez un bordereau de quittancement, il est indispensable de vérifier qu'aucune erreur de saisie n'a été commise.

Pour cela, à l'issue de la saisie du bordereau, vous sélectionner cette option.

Vous obtenez alors l'écran de sélection :
Vous pouvez imprimer l'ensemble des quittances, les quittances prélevées, les quittances non prélevées.

Vous pouvez demander toutes les quittances saisies de la période, ou seulement les quittances Terme, les quittances comptant.

Vous indiquez la période concernée : un mois comptable, ou deux dates.

Si vous précisez les dates, vous indiquez si ce sont les dates d'échéance, ou les les dates de saisie.

Vous pouvez demander l'état pour l'ensemble des compagnies (une entité), pour une compagnie (vous sélectionnez la compagnie). L'état peut etre ventilé par apporteur (sélectionnez les apporteurs), ou réduit aux quittances d'un apporteur( sélectionnez l'apporteur).

Vous pouvez aussi réduire l'état aux quittances d'un contrat groupe.(sélectionnez le contrat groupe)

Clliquez sur Imprimer :

e-ASSUR imprime la liste des quittances saisies selon les critères que vous avez précisés :

- Quittances d'encaissement confiees

- Quittances d'encaissement non confiees

- Ristournes confiees

- Ristournes non confiees

Puis il vous propose de rechercher les quittances manquantes : si vous cliquez sur Chercher, le logiciel va rechercher dans les contrats les quittances qui auraient du etre emises pour la periode, verifie si cette quittance a ete emise . Si ce n'est pas le cas, il insere le contrat dans la liste.

Cette liste d'anomalie peut indiquer des quittances effectivement manquantes, ou bien des erreurs de saisie dans le contrat concerné.
