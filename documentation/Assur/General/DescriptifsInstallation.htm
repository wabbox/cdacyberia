Liste des programmes d'exportation disponibles

Pour installer un descriptif d'exportation, vous devez procéder en trois étapes : Récupérer le fichier d'installation nommé EDI_XXX.TXT  (XXX représente le nom du document. Par exemple :EDI_envoi de fonds.TXT ) et procéder à son installation. Vous devez ensuite vérifier que l'installation s'est bien faite. Enfin, si vous désirez créer un fichier Excel, vous devez créer dans le répertoire EDI un modèle de document dans lequel les informations seront déversées

1°)Première étape : Récupérer le fichier d'installation 

Sur l'écran de gestion des exports, cliquez sur le bouton Télécharger, ou bien cliquez sur le lien Liste des Programmes disponibles. La liste des programmes que vous pouvez télécharger est affichée. Cliquez avec le bouton droit de la souris . Sélectionnez l'option Enregistrer la cible sous ... et enregistrez le fichier dans le répertoire EDI de votre disque s'il existe, sinon dans tout répertoire de votre choix.

Si le programme dont vous avez besoin ne figure pas dans la liste, CDA vous l'aura envoyé en pièce jointe d'un e_mail

Vous devrez alors récupérer dans votre courrier électronique le fichier que CDA vous a adressé en pièce jointe, et l'enregistrer sur le disque dans le répertoire de votre choix. Ce fichier s'appelle EDI_XXX.TXT

Pour cette opération, voir : Enregistrer des pièces attachées

 

Lorsque le fichier est sur le disque, dans le répertoire de votre choix, cliquez sur le bouton Importer

Le logiciel vous demande de localiser le fichier EDI_XXX.TXT :

Sélectionnez le fichier concerné et cliquez sur Ouvrir.

 
 - Vérifiez dans la liste des descriptifs qu'une nouvelle ligne est bien apparue
 - Créez votre modèle Excel si nécessaire
