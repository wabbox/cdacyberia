ANALYSE D'UN DOCUMENT EXCEL

Documentation Generale produits

Sommaire Documentation e_Assur

Sommaire paramètres
Cette analyse permet d'importer automatiquement des flottes, des listes d'adhérents à un contrat collectif qui vous sont fournis par vos clients en format excel. Il y a pour cela 2 Etapes :

A) Etape 1 : Analyse du Document Excel, pour savoir où se trouvent les informations

B) Etape 2 : il faut réaliser l'importation

ETAPE 1 :

1°) Ouvrir le document Excel.  Noter les numéros de première ligne et première colonne des titres.
Refermer le document.

2°) Sur Assur,  en utilitaires ----> Importation de fichiers.
Cliquer sur Nouveau E
- Nom du programme : donner un nom à votre programme exemple : contrats
- Nom du fichier interne : récupération_test
- Norme : XLS
- Procédure à executer : si_test
- Ligne de titre : N° de la ligne du tableau où se trouvent les titres de colonnes (noté en 1°)
- Colonne de titre : N° de la première colonne de titres

Cliquer sur Parcourir. Sélectionner le fichier Excel à analyser et cliquer sur ouvrir.


Puis cliquer sur Analyser.   Le logiciel Excel se lance.  Attendre la fin de l'alerte "Fin de l'analyse".
Cliquer sur continuer
La liste des champs disponible est affichée.
Cliquer sur quitter.

Sur l'écran, gestion des transferts de données, sélectionner à droite votre programme (contrats dans l'exemple ci-dessus)  puis cliquer sur Executer.
Attendre l'alerte  "Fin de l'import"

Cliquer sur voir test pour vérifier que la lecture de feuille Excel s'est faite correctement.
