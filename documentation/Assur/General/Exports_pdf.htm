Sommaire Utilitaires
 

- Dans Assur, les éditions standards se font à partir du bouton états standards - exécuter - sélection de l'imprimante et impression des documents.

Ors, vous pouvez avoir besoin d'envoyer cet état, par mail, en pièce jointe, en format pdf.
exemple :  liste des sinistres d'un client à lui adresser par mail.

Pour cela vous devez disposez de pdf creator. 
Contrôler dans la liste des imprimantes que vous avez bien l'icone de  "l'imprimante" Pdf, sinon le télécharger.

Lorsque la liste est affichée,  cliquez  envoyez vers   cocher  états standards, cocher pdf, donner un nom à votre fichier avec l'extension .pdf , cocher envoyer à l'adresse et l'e mail du client apparait, saisir votre texte éventuellement et cliquez sur executer.  Le dialogue d'impression est affiché, sélectionner l'imprimante pdf et cliquez OK puis enregistrer votre fichier, dans le répertoire voulu.
Attention le nom doit être identique à celui que vous avez saisi dans la rubrique  sous le nom

Le logiciel vous affiche votre document en pdf.  Fermer le document.  Un message vous informe soit
Autoriser l'envoi du fichier  soit interdire.   Si le document pdf est ouvert,  il ne pourra pas être envoyé.
Pensez bien à le fermer avant d'autoriser l'envoi.

Exemple : Envoyer la liste des sinistres d'un contrat à un client.

Sur la fiche contrats - cliquez sinistres -  Envoyer vers - Cocher états standards - cocher pdf
Controler le répertoire dans lequel sera enregistré le fichier.  Le modifier si nécessaire en cliquant sur l'icone du dossier   - donner un nom au fichier     sinMrDURAND.pdf
Cocher envoyer à l'adresse :  contrôler l'adresse mail affichée   - Saisir son texte Executer
Le dialogue d'impression apparait   Séléctionner l'imprimante pdf  OK  OK  et enregistrer le fichier
(par défaut, le nom qui apparait  est sinistres.pdf   le renommer sinMrDURAND.pdf) OK
Enregistrer ce fichier dans le même répertoire qu'indiqué plus haut.  Le fichier  en format pdf est affiché à l'écran.  Fermer le document  et au message  répondre  Autoriser.  Le mail avec la pièce jointe est envoyé.
