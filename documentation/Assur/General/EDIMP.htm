Sommaire Paramètres

Paramètres --->Utilitaires --->Importation de fichiers --->EDI Mutuelle du Poitou
1 - PREMIERE UTILISATION DE L'EDI

Si vous lancez l'EDI pour la premiere fois, vérifiez que tous les paramétrages ont ete effectués correctement.

  • Vérifiez que tous les clients Mutuelle du Poitou ont bien le Numero de sociétaire renseigné dans la zone Numero Compagnie
  • Vérifiez en gestion des Compagnies que la correspondance entre les catégories de la compagnie et vos branches et sous branches a été paramétrée
  • Vérifiez en gestion des branches que la correspondance entre les garanties et extensions de garantie de la compagnie et vos garanties a été paramétrée
  • Vérifiez que les descriptifs de fichiers sont bien présents en gestion des transferts de données, et que vous avez bien indiqué le nom du fichier EDI Mutuelle du Poitou

Dans la liste de droite, vous devez avoir les lignes :

  • Echo Mutuelle du Poitou
  • Echo Mutuelle du Poitou Cients
  • Echo Mutuelle du Poitou contrats
  • Echo Mutuelle du Poitou Quittances
  • Echo Mutuelle du Poitou QuittancesP
  • Echo Mutuelle du Poitou Sinistres
  • Echo Mutuelle du Poitou MVT sinistre

Selectionnez dans la liste la ligne Echo Mutuelle du poitou, cliquez sur le bouton Modifier.

Vérifiez que la zone Nom du fichier externe est bien renseignée : vous devez avoir : MVTxxxx.txt où xxxx est votre code agent.

Cliquez sur Validation.

Cliquez sur Quitter

Une fois que ces informations ont été correctement renseignées, elle n'ont plus à être modifiées.

2 - La recupération du fichier EDI

Allez chercher sur le serveur Mutuelle du Poitou le fichier MVTxxxx.txt.

Si vous êtes en monoposte, placez le dans le repertoire c:\assur\

Si vous êtes en multiposte, placez le sur un poste client, dans le repertoire c:\ACI\Programs\4DCli606\
3 - L'execution de la procedure

Allez en : Paramètres --->Utilitaires ---> Importation de fichiers --->EDI Mutuelle du Poitou

Vous obtenez l'ecran suivant :

Vous pouvez choisir d'importer toutes les données, (toutes les options sont cochées), ou de n'en importer qu'une partie (décochez les lignes correspondant aux données que vous ne désirez pas importer).

La date de dernier import vous permet de vérifier si vous n'avez pas commis d'erreur dans la sequence de vos imports.

4 - Le controle

Après avoir effectué l'importation, allez en Utilitaires - Importation de fichiers, vous verrez dans la colonne importation le nom et la date de votre Importation.

Pour visualiser ce qui a été importé, sélectionnez votre importation et cliquez sur le bouton Voir. Vous verrez apparaître un tableau avec la liste de tout ce qui a été récupéré.

 

Dans la colonne fonction, le type de la fonction : client - contrat - quittances - sinistres....

Dans la colonne opération, les opérations effectuées : emission ou encaissement de quittances, saisie ou clôture sinistres....

La colonne Anomalie vous permettra de diagnostiquer, rapidement le problème rencontré lors de cette importation.

Vous pouvez imprimer cette liste des opérations recupérées.

Au fur et à mesure des importations, vous aurez dans la colonne de gauche, Importation, une liste de toutes vos importations.  Pour conserver,  à l'affichage,  uniquement les dernières opérations, sélectionner celle qui doit disparaitre et cliquer sur  EPURER .

Le bouton SUPPRIMER : NE JAMAIS UTILISER CE BOUTON SAUF sur indication de CDA.
