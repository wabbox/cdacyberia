Sommaire Documentation e_Assur

Sommaire Paramètres

Paramètres --->Utilitaires --->Importation de fichiers --->EDI GAN

1) - Pour la procédure d'installation de l'EDI, cliquez ici

2 - La recupération du fichier EDI

Allez chercher sur le serveur GAN EUROCOURTAGE les fichiers EDI du jour

Placez les dans le répertoire EDI_GAN

 
3 - L'execution de la procedure

Allez en : Paramètres --->Utilitaires ---> EDI GAN

Vous obtenez l'ecran suivant :

Vous pouvez choisir d'importer toutes les données, (toutes les options sont cochées), ou de n'en importer qu'une partie (décochez les lignes correspondant aux données que vous ne désirez pas importer).

 

Pour chaque case cochée, cliquez sur parcourir pour localiser le fichier EDI correspondant dans le répertoire EDI_GAN

Si vous laissez coché Quittances, vous devez indiquer la date de comptabilisation, le mois comptable, et l'exercice.

Cliquez sur Validation

Le programme vous informe lorsque l'importation des données est terminée
4 - Le controle

Après avoir effectué l'importation, allez en Utilitaires ---> Importation de fichiers, vous verrez dans la colonne Historique le nom et la date de votre EDI

Pour visualiser ce qui a été importé, sélectionnez votre EDI et cliquez sur le bouton Voir. Vous verrez apparaître un tableau avec la liste de tout ce qui a été récupéré.

 

Dans la colonne fonction, le type de la fonction : client - contrat - quittances - sinistres....

Dans la colonne opération, les opérations effectuées : emission ou encaissement de quittances, saisie ou clôture sinistres....

La colonne Anomalie vous permettra de diagnostiquer, rapidement le problème rencontré lors de cette importation.

Vous pouvez imprimer cette liste des opérations récupérées.

Au fur et à mesure des importations, vous aurez dans la colonne de gauche, Importation, une liste de toutes vos importations.  Pour conserver,  à l'affichage,  uniquement les dernières opérations, sélectionner celle qui doit disparaitre et cliquer sur  EPURER .

Le bouton Détruire : NE JAMAIS UTILISER CE BOUTON SAUF sur indication de CDA.
