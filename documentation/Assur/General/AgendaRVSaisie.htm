Consultation de l'agenda Rendez-vous

L'agenda rendez vous permet de gérer les rendez vous des collaborateurs du cabinet

 
Vous pouvez programmer des rendez vous de trois manières :

Soit à partir de l'écran Contacts des fiches Client, Contrat, Sinistre, soit par l'option du menu Agenda : Saisir Rendez vous

 
- La date agendée est celle du rendez vous

- L'émetteur et le destinataire :si vous programmez un rendez vous pour vous meme, l'émetteur et le destinataire sont les memes, sinon précisez qui est le destinataire : celui qui a rendez vous

- Client : ces informations sont remplies automatiquement si vous saisissez à partir de la fiche contact, sinon vous pouvez les saisir.

- priorité : vous pouvez indiquer le niveau de priorité de ce rendez vous

- si vous cochez Privé, le rendez vous sera affiché, mais sans l'objet.

- L'objet : c'est un résumé explicite du rendez vous, qui sera affiché dans les listes des rendez vous

- La zone de texte vous permet d'expliciter le motif du rendez vous

- Le bouton Taches vous permet d'accéder à l'agenda des taches.
