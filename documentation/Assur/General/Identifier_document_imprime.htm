IDENTIFIER L'UTILISATEUR

QUI IMPRIME UN DOCUMENT

Identifier l'utilisateur qui a imprimé un document suppose :

1°) Que la connexion au logiciel se fait en Utilisateur et non en Super Utilisateur

Pour paramétrer les Utilisateurs, voir procédure Q dans la documentation Utilisateur.

Rappel : Documentation téléchargeable sur le site de CDA dans la partie Documents à télécharger.

2°) Que le Document imprimé contienne une partie en traitement de textes.

Si ces 2 conditions sont remplies, dans la partie traitement de textes, vous pourrez indiquer :

- soit le nom de l'utilisateur : Pour cela insérer la variable <>utili

Menu Base - Option : Insérer une commande 4D)

- soit la signature du collaborateur. Pour cela, insérer la variable <>v_sign

Menu Base - Option Insérer une commande 4D
