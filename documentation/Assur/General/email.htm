Sommaire Paramètres

Vous pouvez envoyer des e_mails à vos différents interlocuteurs : clients, compagnies, experts...

Pour cela, vous devez avoir au préalable :

1) Paramétré votre adresse e_mail et l'adresse de votre serveur SMTP dans vos paramètres généraux

2) Saisi pour vos interlocuteurs leur adresse e_mail

1) Paramétrage des comptes e_mail

Pour paramétrer : allez en paramètres généraux, double cliquez sur votre entité, cliquez sur le bouton comptes e_mail

Définissez vos préférences générales

Puis, en double cliquant sur la liste des comptes, créez vos comptes e_mail.

Vous pouvez créer autant de comptes que vous voulez, l'essentiel est que l'un d'entre eux soit défini comme compte par défaut

Vous pouvez créer des comptes pour chaque collaborateur : allez en gestion des collaborateurs, sur la fiche du collaborateur, cliquez sur le bouton Paramètres, et vous créer le ou les comptes e_mail du collaborateur.

 

2) Préparer un e_mail

Pour envoyer un e_mail, allez sur la fiche de l'interlocuteur, cliquez sur le bouton Email

Vous arrivez sur le dialogue d'envoi d'un e_mail :

- zone De : l'adresse e_mail du compte par défaut est affichée . Vous pouvez en choisir une autre

- zone Destinataire : l'adresse du destinataire est affichée

Vous pouvez la modifier : par exemple, si vous venez du client, c'est l'adresse du client qui est affichée. Si vous désirez envoyer un courrier à l'apporteur de ce client, vous sélectionnez dans la zone de défilement "Apporteur". C'est l'adresse de l'apporteur qui est affichée.

- zone Copie : vous pouvez envoyer une copie de votre mail à plusieurs destinataires. Il vous suffit de saisir leurs adresses séparées par un point-virgule.

- zone pièce jointe : vous pouvez joindre à votre e_mail des pieces jointes : vous cliquez sur le bouton Pièce jointe et vous sélectionnez le fichier à envoyer autant de fois que nécessaire.

- zone message : par défaut, le message est en format texte brut . Si vous désirez envoyer le message en texte formaté, vous cliquez sur le bouton Html . Vous pouvez toutefois insérer des liens.

S'il s'agit du lien sur un site , vous le saisissez sous la forme : http://www.cda-informatique.fr

S'il s'agit d'une adresse e_mail vous la saisissez sous la forme : mailto:support@cda-informatique.fr

- Bouton Html : il vous permet d'envoyer un message HTML dont le modèle a été au préalable préparé en 4D Write

 

Lorsque vous cliquez sur ce bouton, le logiciel vous demande d'identifier le modèle à utiliser

Vous sélectionnez le modèle et vous cliquez sur Validation.

Si vous ne désirez pas utiliser de modèle, cliquez directement sur Validation.

Le document traitement de texte sera envoyé au destinataire sous format HTML

 
Si dans votre document vous désirez insérer des liens, allez dans le menu Insertion de 4D Write et sélectionnez l'option Lien Hypertexte

L'écran de paramétrage du lien s'affiche : cochez Ouvrir une URL

Pour un lien sur un site , saisissez dans les deux zones de saisie l'adresse http sous la forme : http://www.cda-informatique.fr

Pour une adresse e_mail, saisissez dans la zone Texte à afficher l'adresse sous la forme : support@cda-informatique.fr et dans la zone Ouvrir une URL l'adresse sous la forme :  mailto:support@cda-informatique.fr

 

Pour une image :

Si vous voulez envoyer une image, il faut utiliser le protocole http

1) Vous mettez l'image sur un site accessible par le web

2) Dans votre courrier vous placez le curseur a l'endroit où vous voulez que l'image apparaisse.

3) Allez dans le menu insertion, option Expression Html et saisissez l'adresse de l'image sur le site sur me modèle suivant :

<font class="f1"><img width=117 height=117 alt="" src="http://www.cda-informatique.fr/images/logo_cda.gif"></font><font class="f2">votre logo</font>
</p>

 

3) Envoi de l'e_mail

Vous pouvez envoyer votre e_mail tout simplement en cliquant sur le bouton envoyer, ou l'envoyer par l'intermédiaire de Outlook en cliquant sur le bouton Outlook. Attention, dans ce cas, les pièces jointes ne sont pas transférées à Oulook.

Si votre message est en texte brut :  cliquez sur le bouton, tout est transféré à Oultlook sauf les pièces jointes.

Si votre message est en 4D Write : le texte du message n'est pas transféré : arrivé sur l'écran Outlook, allez dans le menu Edition et sélectionnez l'option Coller
