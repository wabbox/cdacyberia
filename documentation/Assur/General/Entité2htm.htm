OPTION : Paramètres                            Menu : Courriers                            Option : Gestion des courriers

CREATION  DE   L'ENTETE  EN  FONCTION  DE  L'ENTITE
Cette partie concerne les utilisateurs qui gèrent plusieurs entités.

Dans ce cas, l'entête du courrier doit correspondre à l'entité concernée.

En gestion des courriers, Menu Base - Option Insérer une commande 4D

 

vnomcomm                      Nom commercial

vadresse                         adresse

v_codeville                     code postal et la ville

v_rc                                n° siret

v_tel                                téléphone

v_fax                               fax
