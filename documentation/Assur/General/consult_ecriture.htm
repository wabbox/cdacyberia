CONSULTATION D'UNE ECRITURE
Fonction :   Comptabilité                Option :  Compta Gle          Sous option :  Consulter une écriture

Vous pouvez toujours consulter une écriture, soit par

I - CLIENT CONSULTER

Appelez votre client. Sur la fiche, cliquez sur le bouton Compte. La liste des opérations apparaît. Sélectionner la ligne qui vous intéresse. Cliquez sur le bouton écriture. Vous verrez ainsi les comptes comptables qui ont été mouvementés par cette écriture ainsi que le numéro de l'écriture. Puis cliquer sur validation. Vous êtes de retour sur le compte client.

ATTENTION : Il s'agit d'une consultation - Ne faites pas de modification dans l'écriture.

II - COMPTABILITE GENERALE, CONSULTER UNE ECRITURE

Vous obtenez l'éditeur de formules.

Vous construisez votre requête en précisant

Colonne 1 : elle permet de préciser les critères de tris souhaités.

Liste des plus utilisés : agence, montant, n° de police, n° de client, compagnie, exercice, numer (n° d'écriture).

Colonne 2 : celle-ci permet de préciser le sens de la recherche. Egal, supéreur,différent...

Rubrique valeur  : permet d'inscrire la valeur demandée.

Exemple    Date écriture est égale au 01/01/2006
 

Si votre requête nécessite plusieurs critères,  : cliquez sur Ajouter et préciser votre second critères de recherche.
et ainsi de suite.

Exemple : Vous recherchez l'écriture 1706 pour l'entité ""courtage""

Numéro_ecriture  est égal à        1706   (cliquez ajouter)
et  entité             est égal à      courtage
