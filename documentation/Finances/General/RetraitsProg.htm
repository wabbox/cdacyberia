Simulations : Retraits programmés

Pour effectuer cette simulation, vous devez :

1) Avoir créé un type d'opération : Plan d'épargne (Colonne : Environnement; Menu Types d'opérations; option Gestion des types d'opérations

2) Avoir créé un état pour ce type d'opération.

L'ecran est divisé en quatre zones :

1) Le menu déroulant en haut d'écran vous permet de selectionner le produit si vous en avez paramétré plusieurs.

Ce paramétrage est essentiel pour pouvoir tenir compte des valeurs de rachat éventuelles, ainsi que des frais de souscription et des frais de gestion.

Si vous n'avez pas paramétré de produit, la simulation ne tiendra compte d'aucun des paramètres ci-dessus.

2) L'acquéreur : Si vous faites une simulation à partir d'une fiche client, les différentes zones sont déjà complétées.

Sinon, vous devez les compléter vous_meme. Le revenu imposable doit etre saisi net d'abattements. Le nombre de parts doit etre calculé au préalable si nécessaire. La TMI et l'impôt sont calculés automatiquement.

3) Les caractéristiques de l'opération :

Les versements sur le plan peuvent ,être unique et/ou périodiques.

Le revenu demandé est exprimé annuellement.

4) Les hypotheses : Le taux de revalorisation correspond à l'hypothèse qui est faite sur l'évolution de la valeur du placement. Le taux d'évolution du revenu correspond à l'augmentation annuelle des prélèvements sur ce compte. Le taux d'augmentation des revenus autres correspond à l'évolution annuelle prévisible des revenus de type salaires, pensions, retraites ... que votre client va percevoir par ailleurs.
