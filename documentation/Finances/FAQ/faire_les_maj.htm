FAIRE LES MISES A JOUR   FINANCES TOP  POUR WINDOWS SUR INTERNET
 

1ère étape : Téléchargement

Aller sur le site www.cda-informatique.fr.  http://www.cda-informatique.fr  Vous arrivez sur la page d’accueil.  Cliquer sur mise à jour.

Sur la nouvelle page, apparaît la liste de toutes les versions Assur.

 Cliquer sur le lien Mise a jour Finances et e_care .Sur la page suivante, cliquez sur Monoposte V7 pour PC et enregistrez dans le dossier FinancesV7  qui est sur votredisque. 

Une fois le téléchargement terminé,  quittez Internet.
 

 

2ème étape : Décompactage

VERIFIEZ QUE VOUS AVEZ BIEN QUITTE :

-          Pour les versions monopostes : le logiciel  Finances TOP

-          Pour les versions multipostes

o        Sur les postes clients : quitter le logiciel  Finances Top

o        Sur le poste serveur :  quitter 4 D serveur

Après avoir quitté Internet, aller sur le poste de travail, sur C :  et double cliquerez sur le dossier  FinancesV7

Vous avez à l’intérieur le fichier que vous venez de  télécharger. FinMonV7.exe.Vérifiez la date du fichier.

Double cliquez dessus.

Un message en anglais – cliquer sur OK –  Un nouvel écran avec à gauche  la rubrique C:\FinancesV7 Vérifiez que le décompactage se fait bien dans le bon dossier et sur le bon disque et cliquez sur unzip.

Un message vous informe que les fichiers ont été décompactés avec succès

 La mise à jour est faite. Cliquez sur Close. Relancez le logiciel Finances.
