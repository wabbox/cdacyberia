Utilisation de 4DUTIL 6.7

 

Preliminaires :

    Effectuer une copie de sauvegarde du fichier assur2v7.4DD

    Dans le dossier Poste de travail c: \ Maintenance Assur

Mono poste :

    Ouvrir le poste de travail, aller sur le disque c:

   Double- Cliquer sur le répertoire AssurV7, puis sur le répertoire 4d_utils_674 et enfin double cliquer sur le programme 4DTools.exe.

Multi postes :

    Ouvrir le poste de travail, aller sur votre disque c:

    Double-Cliquer sur le répertoire 4D,  ou Program files\4D  puis sur le répertoire Outils, puis 4D_utils_674 et enfin double cliquer sur le programme 4DTools.exe.

 Une fois le programme 4Dutil lancé, une fenêtre Ouvrir s'affiche, sélectionner alors le fichier assur2v7.4DC ( poste de travail \ c: \ AssurV7 \ ) et entrer le mot de passe super_utilisateur.

Un écran gris s'affiche avec des onglets : cliquez sur l'onglet Maintenance

 

Compacter :

4D Util va recopier le fichier de données Assur2V7.4DD en le compactant.

Si vous sélectionnez cette option, 4D Util vous présente une fenêtre d'enregistrement de fichier : Créer un fichier de données.Il propose de nommer le nouveau fichier Asur2V72.4DD et de l'enregistrer dans C:\AssurV7\.

Cliquez sur enregistrer.

Le compactage commence, son déroulement est décrit par une suite de messages.

 

 

Vérifier et Réparer :

 4D Util va recopier le fichier de données Assur2V7.4DD en le réparant :

    Dans la fenêtre suivante cocher : Récupération par analyse des Marqueurs de fiche et cliquer sur OK.  

    Dans la nouvelle fenêtre nommée " Créez un fichier de données "  la rubrique :  Nom du fichier doit inscrire automatiquement assur2v7_2.4DD. Cliquer  alors sur Enregistrer.

    Une fois la récupération terminée, quitter le programme 4Dutil.

Comparer :

    Il faut maintenant comparer la taille des deux fichiers assur2v7.4DD et assur2v7_2.4DD ( sélectionner le fichier en cliquant une fois puis bouton droit, rubrique propriété. La taille du fichier apparait dans cette fenêtre. Faire de même pour le second fichier )

    Les fichiers se trouvent dans : Poste de travail \ c: \ AssurV7

Vérifier

     Il faut également vérifier le nombre des fiches par éléments de assur2v7.4DD.

    Ouvrir le logiciel e-Assur dans le menu Utilitaire, cliquer sur " A propos de assur2 ", noter les résultats obtenus et quitter le logiciel.

    Renommer ensuite le fichier assur2v7.4DD en Xassur2v7.4DD.

    Et renommer le fichier assur2v7_2.4DD en assur2v7.4DD.

    Ouvrir à nouveau le logiciel e-Assur dans le menu Utilitaire, cliquer sur " A propos de assur2 ", noter les résultats obtenus et quitter le logiciel.

    Comparer les résultats pour obtenir le nombre de fiches perdues. Ces fiches ( ou écritures ) sont à retrouver et à ressaisir.

Attention :

    Le fichier de donnée que vous souhaitez conservez et utiliser doit toujours s'appeler assur2v7.4DD.

    Si vous ne voulez pas utiliser le nouveau fichier pensez à supprimer ou à déplacer le nouveau fichier assur2v7.4DD et renommer le fichier Xassur2v7.4DD en assur2v7.4DD.
