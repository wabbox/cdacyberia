Sommaire general e_Care

1) L'écran d'accueil
Cet écran comporte 3 zones :
  • à gauche la liste des fichiers de votre ordinateur qui devront être sauvegardés

(Pour l'instant, vous devez indiquer au logiciel tous les fichiers qu'il doit sauvegarder, dans une version ultérieure, vous pourrez indiquer des répertoires entiers)

  • en haut les différents onglets permettant le paramétrage du logiciel
  • à droite les boutons permettant d'effectuer les opérations de sauvegarde

2) Les fichiers à sauvegarder
Sur l'exemple, un seul fichier sera sauvegardé : le fichier de données de AssurV7

a) Ajouter un fichier à sauvegarder : Pour ajouter un fichier, cliquez sur le bouton Ajouter à la liste

Vous obtenez l'écran d'exploration qui vous permettra de désigner les fichiers à sauvegarder.
Dans ce cas, on demande de sauvegarder le document Microsoft Word "Calcul Matriciel", en plus des données e_Assur

La liste des fichiers à sauvegarder devient alors :
b) Supprimer un fichier de la liste

Sélectionnez le fichier a supprimer dans la liste de gauche, cliques sur le bouton Supprimer de la liste

3)L'onglet Paramètres
Il vous permet d'indiquer la destination de la sauvegarde : vous pouvez sauvegarder sur un disque externe, de type ZIP ou Jazz, vous pouvez sauvegarder sur un ordinateur du réseau, vous pouvez enfin sauvegarder sur notre serveur FTP via Internet.

Vous pouvez déclencher cette sauvegarde manuellement, à la demande, ou demander à e_Care de la déclencher à jour et heure fixe.
La boite à cocher Sauvegarde Programmée : Si vous cochez cette case, le e_Care se déclenchera à jour et heure fixes. Pour que cela fonctionne, vous devrez aller sur l'onglet Programmation pour indiquer la fréquence (jour ou semaine), et l'heure de la sauvegarde .Vous devrez par ailleurs vous assurer que l'ordinateur sera en route à l'heure indiquée.

La boite à cocher Serveur FTP : Si vous cochez cette case, la sauvegarde se fera sur notre serveur FTP, via Internet. Vous devez, pour que cette solution fonctionne, avoir souscrit un abonnement.

La rubrique Répertoire de travail : Avant d'effectuer la sauvegarde, e_Care compacte vos données, et stocke le fichier ainsi compacté dans le répertoire de travail. Par défaut, ce répertoire est défini comme étant sur le même disque que l'application e_Care. Ne changez la valeur de cette rubrique que si vous n'avez pas assez de place sur ce disque .Utilisez alors le bouton Parcourir pour localiser un autre répertoire sur un autre disque.

La rubrique Nom du fichier Archive : C'est le nom du fichier dans lequel e_Care va sauvegarder vos données. Vous pouvez donner à ce ficher le nom que vous désirez.

La rubrique Destination de la sauvegarde : Si vous n'avez pas coché la case Serveur FTP, c'est que vous désirez sauvegarder vos fichiers sur un disque externe, ou sur un autre ordinateur du réseau. Dans ce cas, vous devez localiser la destination de votre sauvegarde à l'aide du bouton Parcourir.

3)L'onglet Serveur FTP
Cet écran vous permet de paramétrer les informations indispensables pour effectuer vos sauvegardes sur notre serveur FTP. Normalement, vous ne devez pas modifier ces informations.

3)L'onglet Programmation
Cet écran vous permet de paramétrer la fréquence des sauvegardes .Ci-contre, e_Care est paramétré pour sauvegarder tous les jours à 10h23.
