Les vérifications préalables
Sommaire

1) L'ordinateur : Vous allez traiter essentiellement des images. Vous devez donc avoir un ordinateur performant : Un processeur rapide, une mémoire de 256 Mo minimum

2) Les périphériques : Vous devez disposer d'un appareil photo numérique et d'un graveur de CD

3) Les logiciels : Av@l peut piloter le logiciel d'acquisition de photos, et le logiciel de gravure du CD;

Vous devez connaître le nom de ces logiciels et leur emplacement sur le disque.

Si vous désirez utiliser ces possibilités, notez le chemin d'accès à ces logiciels, afin de pouvoir les indiquer en paramètres généraux.

4) Le navigateur : L'aide en ligne est accessible par l'intermédiaire de votre navigateur Internet. Il faut donc, là aussi, que vous connaissiez l'adresse de votre navigateur afin de l'indiquer en paramètres généraux.. Vous pourrez ainsi bénéficier de l'aide en ligne sur tous les écrans.
