Saisie d'un objet
Sommaire

L'écran de saisie d'un objet vous permet de saisir les informations le concernant : la catégorie à laquelle il appartient, par l'intermédiaire d'un menu déroulant hierarchique. Vous pouvez gérer le contenu de ce menu déroulant en Gestion des énumérés. Vous saisissez aussi son nom, sa description sa date d'acquisition, la valeur d'achat et la valeur actuelle (en euros). Si vous disposez d'une facture pour cet objet, cliquez sur la boîte à cocher Facture d'achat. De même, si l'objet a fait l'objet d'une expertise, cochez la boîte à cocher correspondante.

Vous pouvez par ailleurs saisir des observations. Pour affecter une photo à cet objet, cliquez sur Choisir Photo.
Le bouton Quitter vous permet de revenir à la liste

Le bouton Courrier vous permet d'envoyer un courrier à votre client concernant ce objet

Le bouton Reclamation vous permet d'imprimer une fiche destinée à la compagnie en cas de sinistre

Le bouton Validation vous permet d'enregistrer cet objet dans le dossier du client, et d'en saisir éventuellement un autre

Le bouton Supprimer vous permet de supprimer un objet du dossier du client.
