Objets
Sommaire

Lorsque vous avez cliqué sur le bouton Objets de la première page du dossier d'un client, vous arrivez sur la page donnant la liste des objets déjà saisis pour ce client

Le bouton Courrier vous permet d'envoyer au client un courrier

Le bouton Imprimer liste vous permet d'imprimer la liste des objets appartenant au client.
Pour voir, modifier, ou supprimer un objet, vous double-cliquez sur sa ligne

Pour Ajouter un objet, vous double-cliquez sur une ligne vide, ou vous cliquez sur Ajouter Objet.
