Saisie d'un Dossier
Sommaire
Vous saisissez les informations concernant le client.

Le numero est attribué automatiquement par le logiciel, Mais vous pouvez le modifier.

Le nom fait l'objet d'un controle d'homonymes. Si vous tapez Dupond, et que Dupond existe déjà, le logiciel vous propsera la liste des Dupond. Cela vous permettra de vérifier s'il s'agit bien d'un nouveau client ou non.
S'il s'agit bien d'un nouveau client, vous cliquez sur continuer. Par contre, s'il s'agit d'un client déjà saisi, sélectionnez le dans la liste, il sera chargé.
Les rubriques telles que le titre(la civilité),l'état civil, la nationalité, la profession, sont saisis grâce à un menu déroulant. Vous cliquez sur la flèche, à droite, vous sélectionnez la valeur.

Si vous désirez changer le contenu de ces listes, allez en Gestion des énumérés.
Apres avoir saisi les informations générales concernant le client, vous devez :

1) Transférer les photos qui ont été prises dans le répertoire du client (bouton Importer photos)

2) Créer les objets et leur affecter ces photos (bouton Objets)

3) Créer la base de données propre à ce client, qui sera transférée sur CD-ROM (bouton Exporter)

Les autres boutons vous permettent :

- d'imprimer le dossier complet du client (bouton Imprimer)

- d'envoyer un courrier au client si vous disposez de l'option traitement de texte .( bouton Courrier)

- de detruire ce client avec tout ce qui le concerne (bouton Detruire)
