Page d'accueil
Sommaire
Lorsque vous lancez le logiciel, vous arrivez sur la page d'accueil qui vous permet de naviguer dans tout le logiciel. Les 5 boutons de gauche concernent la gestion de vos dossiers clients, les trois boutons de droite vous permettent de gérer les parametres de votre application.

Les Boutons de parametres :

Parametres : ce bouton vous permet de définir l'environnement de l'application

Codes Postaux : ce bouton vous permet de modifier ou enrichir la base de données Codes Postaux

Enumérés : ce bouton vous permet de modifier le contenu des listes déroulantes

 

La Gestion des dossiers clients :

Pour saisir un nouveau dossier, double-cliquez sur une ligne blanche ne contenant aucun client

Pour consulter un dossier d'un client, double-cliquez sur la ligne correspondante.
