LES PARAMETRES GENERAUX DE L'APPLICATION
Sommaire 




  

    Vous pouvez indiquer ici les principaux parametres qui décrivent à votre
    logiciel son environnement de travail
  






  

    

      

        Numero de version et Parametres : sauf exception vous ne
        devrez pas changer ces indications. Elles seront par contre utiles à nos techniciens pour
        connaitre votre version du logiciel.
Chemin Photos :
        Lorsque vous allez décharger les photos de votre appareil numerique, il faut que le
        logiciel puisse les retrouver. Il existe dans le repertoire de l'application un sous
        repertoire appelé Photos dans lequel les repertoires correspondant à chaque dossier
        client seront créés pour stocker les photos. Si toutefois pour diverses raisons vous
        deviez changer cet emplacement, il faudrait indiquer ici le chemin d'acces complet pour
        que le logiciel puisse retrouver les photos. Par exemple, si vos photos sont dans le
        repertoire de l'application de votre appareil photos, vous pourriez avoir : C:\OLYMPUS\Photos.

        
Chemin Appli Photo : Selon les marques et les modeles
        d'appareils, le logiciel qui permet le transfert des photos de l'appareil vers
        l'ordinateur peut se trouver soit dans l'appareil, soit avoir été installé sur le
        disque dur. S'il se trouve sur le disque dur, il peut etre piloté par votre application.
        Dans ce cas il faut indiquer à votre application son nom et son chemin d'acces. L'exemple
        ci-dessus indique qu'il s'agit du logiciel C-W95.EXE dont le chemin d'acces est C:\OLYMPUS\CAMERA95\ .

        
Chemin Appli CD : Lorsque le dossier
        du client est constitué, vous pouvez le transférer sur un CD. Dans ce cas, vous disposez
        sur votre disque dur d'un logiciel qui vous a été fourni avec le graveur. Pour que ce
        logiciel puisse etre piloté automatiquement par votre application, il faut lui indiquer
        le nom et le chemin d'acces. Dans l'exemple ci-dessus, Le logiciel necessaire pour la
        gravure du CD s'appelle cd toast.exe et son chemin d'acces est : C:\Program
        Files\Adaptec\ .

        
Chemin Explorer : L'aide de votre application peut etre
        lue par un navigateur tel Microsoft Internet Explorer ou Netscape. Vous indiquez dans
        cette rubrique le nom et le chemin d'acces de votre navigateur. Dans l'exemple ci-dessus
        on utilise Microsoft Internet Explorer iexplore.exe dont le
        chemin d'acces est C:\Program Files\Plus!\Microsoft Internet\ .

        
Vous pouvez aussi avoir : c:\program
        Files\Internet Explorer\iexplore.exe

        
Chemin Aide : Les documents
        composant cette aide sont stockés dans un repertoire Documentation qui se trouve dans le
        repertoire de votre application. Le chemin d'acces est donc : C:\Expert\Documentation. Si vous désirez que cette aide soit stockée ailleurs, par exemple vous
        ne voulez pas la stocker sur le disque, mais l'utiliser sur le CD d'installation, vous
        indiquerez le chemin D:\Documentation.
