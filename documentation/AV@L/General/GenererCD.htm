Générer le CD Cient
Sommaire

Après avoir saisi la totalité du dossier d'un client dans votre base de données, vous devez créer un fichier qui lui est propre, et graver un CD , opération qui rendra ces informations non modifiables.

La génération se fait en deux temps :

1) Création du fichier client dans un répertoire au nom du client

2) Gravage du CD
1) La création du fichier client :

Lorsque vous cliquez sur Exporter, l'application va créer tous les fichiers nécessaires au gravage du CD client et les placer dans un répertoire dont le nom est composé du numéro du client et de son nom.

Par exemple, s'il s'agit du client 000028 qui s'appelle Dupond, le répertoire s'appellera 000028DUPOND.

Dans ce répertoire seront placés un certain nombre de fichiers, pilotés par une application dont le nom sera, dans notre exemple, 000028DUPOND.EXE. Cest cette application qui devra être lancée pour consulter le dossier du client.

Lorsque la création du fichier Client est terminée, un message vous en prévient. Vous Cliquez sur OK et l'application du client est lancée pour vous permettre de contrôler que tout s'est bien passé.

Vous arrivez alors directement sur la fiche du client.
Vous faites les différents consultations, et si tout est correct, vous quittez. Vous revenez alors à votre base de données. Si vous avez paramétré dans les paramètres généraux le chemin d'accès à votre logiciel de gravage de CD, celui-ci est lancé. et vous pouvez graver le CD.
