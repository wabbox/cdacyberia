Sommaire FAQ e_Assur

Quand on fait un état sur une table, seules les informations stockées dans cette table sont disponibles.

Pour récupérer des informations stockées dans une autre table, il faut créer une colonne supplémentaire, et y adjoindre une ligne de programmation pour aller chercher l'enregistrement correspondant de l'autre table.

 

Exemple: si on veut editer un état sur contrats comportant les colonnes :

[Contrats]Numero_client [clients]nom [contrats]Numerocontrat

On dispose dans la table Contrats du numero de client, et du numero de contrat, mais pas du nom qui se trouve dans la table Clients.

 

Il faut donc :

1- Creer vos trois colonnes

1-Rajouter une colonne : Menu Edition, Ajouter colonne

2-Dire au logiciel d'aller chercher, pour chaque contrat, le nom du client correspondant.

1) Création des trois colonnes de l'état

Double-cliquez sur NUMERO_Client

La colonne correspondante de l'etat est créée

Cliquer sur la croix à gauche de NUMERO_Client

Vous obtenez la liste des champs de la table Clients : double-cliquez sur Nom

La colonne correspondante de l'etat est créée

Cliquez sur le "- " à gauche de NUMERO_client

Les champs de la table Clients sont masqués

Double-cliquez sur NumeroContrat

La colonne correspondante de l'etat est créée

Si vous demandez l'impression de l'état maintenant, la colonne du nom sera vide. Il faut maintenant passer à la seconde étape.

2) Création d'une colonne de programmation

Edition ----> Ajouter Colonne

Tapez Chercher(

Cliquez sur la croix à gauche de NUMERO_client. Les champs de la table Clients sont affichés.

Double-cliquez sur Numero_client de la table Clients

Tapez "="

Cliquez sur le "-" à gauche de NUMERO_client de la table Contrats. La liste des champs de la table Clients est masquée.

Double-cliquez sur NUMERO_client de la table Contrats.

Fermez la parenthèse )

Cliquez sur OK

Pour que cette colonne de programmation ne s'imprime pas, sélectionnez la nouvelle colonne, allez en Edition -------> Cacher

Vous pouvez demander l'édition de votre Etat.
