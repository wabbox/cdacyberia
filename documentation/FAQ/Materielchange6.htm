CHANGEMENT D'ORDINATEUR
I - CHANGEMENT DU POSTE SERVEUR

Si vous voulez changer l'ordinateur équipé du serveur 4D, il faut, pour les utilisateurs en Version 6

1) Copier sur le nouveau serveur les trois dossiers suivants :

- C:\assur                   (de l'ancien serveur)      

- C:\aci                      (de l'ancien serveur)

- C:\Windows\Aci   (de l'ancien serveur) ou C:\Winnt\Aci

Copier ces répertoires par le principe du copier coller.

ATTENTION : LES REPERTOIRES COPIES DOIVENT AVOIR LE MEME CHEMIN D'ACCES.

Sur le nouveau Serveur, il doit donc y avoir :

  - 1  dossiers à la racine C:\ (Assur et Aci)                - 1 dossier dans le répertoire Windows (Aci)

2) Sur le nouveau serveur :

Supprimer les fichiers ASSUR2V6.REX et ASSUR2V6.RES 

(Ces fichiers se trouvent dans C:\Windows\Aci)
POUR LES UTLISATEURS EN VERSION 7   pour le changement du Serveur

Les dossiers sont :   ASSURV7      4D (dans C:\Program Files)  et  4D  dans c:\Windows (ouWinNt)
II - CHANGEMENT DU POSTE CLIENT

Si vous voulez changer l'ordinateur équipé du 4D Client, il faut :

1) Copier sur le nouveau client les deux  dossiers suivants :

- C:\Aci                    (de l'ancien client)

- C:\Windows\Aci   (de l'ancien client)

Copier ces répertoires par le principe du copier coller.

ATTENTION : LES REPERTOIRES COPIES DOIVENT AVOIR LE MEME CHEMIN D'ACCES.

Sur le nouveau Client, il doit donc y avoir :

  - 1 dossier à la racine du disque C:\  (Aci)                          - 1 dossier dans le répertoire Windows (Aci)

2) Sur le nouveau Client :

Supprimer les fichiers ASSUR2V6.REX et ASSUR2V6.RES

(Ces fichiers se trouvent dans C:\Windows\Aci)
POUR LES UTLISATEURS EN VERSION 7 pour le changement du 4D Client

Les dossiers sont  4D (dans Programs Files) et 4D dans Windows ou Winnt

La suppression des fichiers est ASSUR2V7.REX et ASSUR2V7.RES (dans c:\windows\4D\AssurV7) et ASSURV7.4DC (dans c:\windows\4D)  ou winnt
